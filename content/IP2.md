## Course Overview

**Internet Programming and Applications II (CP 311)** is an advanced web development course offered to third-year students in the College of Informatics and Virtual Education (CIVE) at The University of Dodoma. This course builds upon foundational web development concepts to cover advanced topics including server-side programming with PHP, database integration with MySQL, asynchronous JavaScript (AJAX), XML/JSON data formats, web services (SOAP and REST), and modern frontend frameworks like Bootstrap.
