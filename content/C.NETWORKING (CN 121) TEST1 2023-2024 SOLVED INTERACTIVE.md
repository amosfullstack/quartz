**1.** In the TCP/IP model, which layer corresponds to the OSI model's Transport Layer?  
- A. Application Layer  
- B. Internet Layer  
- C. Transport Layer  
- D. Network Access Layer  

> [!Tip]- Answer (Click to show)
> **C. Transport Layer**



**2.** Which layer of the OSI model handles end-to-end communication between applications?  
- A. Transport Layer  
- B. Data Link Layer  
- C. Application Layer  
- D. Presentation Layer  

> [!Tip]- Answer (Click to show)
> **A. Transport Layer**



**3.** Which Ethernet standard utilizes fiber optic cables for transmission?

- C. 1000BASE-T  
- D. 100BASE-T  

> [!Tip]- Answer (Click to show)
> **B. 100BASE-FX**



**4.** What is the maximum cable length for 1000BASE-T Ethernet connections using twisted pair cables?

> [!Tip]- Answer (Click to show)

**5.** What is the purpose of the CSMA/CD (Carrier Sense Multiple Access with Collision Detection) protocol in Ethernet?
