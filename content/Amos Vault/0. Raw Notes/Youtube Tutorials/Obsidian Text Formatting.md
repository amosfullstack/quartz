
# Emphasis

**This is a Bold Sentence
- Use two stars `**` at the beginning of the Sentence

This is a Bold **Word** in a sentence
- Use two stars at the beginning and End of the `**`Word`**`

*This is a italicized sentence
- Use one star at the beginning of the Sentence

This is a italicized *Word* in a sentence
- Use one star at the beginning of the `*`word`*` and at its end

## Task Lists
- [ ] Eating
- [ ] Coding
- Use `- [ ] ` This Creates a tasks List

## Strike through the text

~~I am Going to School

- Use two `~~` signs

## Highlight

==This is the highlighted Text

- Use two Equal Signs ==

## Horizontal Line
1
---
- Use three Dashes `---`

## Block Quotes
- Use the greater than Symbol `>`

## Callouts

> [!info] This Section is Worth Remembering

- Use `> [!info]`

## Hyphen
\- Food
\- Cloths
\- Shelter

- Use `\-` symbols

## Adding a note

> [!Note]
> This is a Note Block

- Use `> [!Note]`
## This is a Warning

> [!Warning]
> This is a Warning Block

- Use `> [!Warning]`

## Tip

> [!Tip]
> This is a tip Block

- Use `> [!Tip]`
