- Git Kranken in GUI tool for git.
- You can create a repositoy.
- Initialize a repository.
- Clone a repository.
- You can view all the commits and the authors of those commits.

###### Steps for using gitkraken
- Open then app and then <span style="color:rgb(255, 0, 255)">open your repository</span>. Select the folder from your pc.
- Inside git kraken you can see the commits, commit message, author of the commit and the data and time the commit was created.
- Click on each commit to see more <span style="color:rgb(255, 0, 255)">details</span> .

Practical:
- Change one file in vscode of that repo.
- The go back to gitkraken.
- View the change.
- Click the file to see the detailed change.
- Click <span style="color:rgb(255, 0, 255)">stage a file</span>, or click <span style="color:rgb(255, 0, 255)">stage all changes</span>.
- Write a commit message before commiting.
- Click <span style="color:rgb(255, 0, 255)">commit</span> to commit changes
