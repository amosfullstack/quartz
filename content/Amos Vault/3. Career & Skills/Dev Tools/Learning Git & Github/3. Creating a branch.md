
___

Steps:
i. First Clone the repository.
```bash
git clone <repo-link>
```

ii. To a see all branches in your git repo
```bash
git branch --all
```

iii. Create a new branch
```bash
git brach mybrach
```
- Upto here we have our new branch localy but not Remotely.

iv. Switching to the created branch
```bash
git checkout mybranch
```
- It time to work on the new branch and push it to github.

v. Open the current repo in to the text Editor cs code
```bash
code .
```
- Make changes to any file, eg the README.md

vi. Add the Modified file to the branch
```bash
git add README.md
```

v. Commit the Changes
```bash
git commit -m "Update the Readme file"
```

vi. To push my New branch and all the changes in it
```bash
git push --set-upstream origin mybranch
```
- The parameter `--set-upstream` is mandatory only the first time, for the next time when you want to push the changes to the remote branch, just use: <span style="color:rgb(255, 255, 0)">git push</span> 
- After the above, the new branch will appear on the github portal.
