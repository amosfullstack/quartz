## Overview
- After viewing the files in the staging area, you can decide to either commit them or unstage them.

What is unstaging?
<span style="color:rgb(0, 176, 80)">Answer</span> : Unstaging basically means to undo the <span style="color:rgb(0, 176, 80)">add</span> command.
Method : We use git Restore.

Git Restore can be used to resotore the files in the working directory and the staging area.

i. Restorig the files in the staging area
```bash
git restore --staged file1

git restore --staged file1.txt file2.js //We can list multiple files

git restore --staged *.js //We can list patterns

git restore --staged . //Restore everything in the staging area
```
 - All the changes that we had in the staging area, will come back to the working directory.

What if a file is newly added in the Staging area.
- if the file is new in the staging area, unstaging it will make the file <span style="color:rgb(0, 176, 80)">untracked</span>.
