Github Codespaces
<span style="color:rgb(255, 148, 148)">Meaning</span> :
- Is a customizable cloud development environment.
- You need just a browser or an IDE that supports Codespaces.

**GitHub Codespaces** is a cloud-based development environment provided by GitHub.
- Think of it as a **VS Code editor + development server in the cloud**, directly connected to your GitHub repository.
- Instead of setting up languages, compilers, and tools on your local machine, you can spin up a preconfigured dev environment inside GitHub with one click.


<span style="color:rgb(255, 148, 148)">Settings</span> :
- You can customize your environment with a cofiguration file directly in your repository.
- You can export the configuration and reuse the settings in multiple repositories and projects.

<span style="color:rgb(255, 148, 148)">Working</span> :
- You can run codespaces in a virtual machine  from two cores upto 32 cores.
- You can create codespce for any branch in your repository and start to work on it.
- Each user can access only their own codespaces and nothin is shared.

<span style="color:rgb(255, 148, 148)">Cost</span> :
- Github codespaces is not for <span style="color:rgb(255, 255, 0)">FREE</span>, but you pay for what you use.
