## Overview
- In VSCode, we have the source control panel, where we can view the changes in our code.
- This is the same as the ouput of the git status command.

Practicals in VS Code:
- Click <span style="color:rgb(255, 0, 255)">changes</span> to see the changes file details.
- Click the + infront of the file to <span style="color:rgb(255, 0, 255)">stage</span> the changes.
- Click the <span style="color:rgb(255, 0, 255)">undo arrow</span>, to undo the changes.
- Add <span style="color:rgb(255, 0, 255)">message</span> on the space give.
- Click <span style="color:rgb(255, 0, 255)">Commit</span> Tick to commit the changes.

See all the Commits of the file:
- Click on the <span style="color:rgb(255, 0, 255)">file</span> itself.
- The at the bottom left conner of vs code, Click <span style="color:rgb(255, 0, 255)">timeline</span> .
