## Overview
- Learn how to search for commits by (author, date, message)
- Learn on how to view a commit and see what files have been changed and how.
- Restore your project to an earlier point. Of the snapshots stored in one of your commits.
- Learn to compare commits to see what files have been changed.
- View the history of a file to see how it evolved from day one, and who has written each line of code.
 - A tool, to find a bad commit that intriduced a bug.

Practicals

i. See <span style="color:rgb(255, 255, 0)">all</span> the commits
```bash
git log 
```
- The commits are listed from the newest to the oldest.

ii. See the <span style="color:rgb(255, 255, 0)">summary </span>of the git log command
```bash
git log --oneline
```

iii. To see all the <span style="color:rgb(255, 255, 0)">files</span> that have been <span style="color:rgb(255, 255, 0)">changed</span> in a certain commit
```bash
git log --oneline --stat
//Include the stat Option

//alse
git log --stat
```

iv. To see the <span style="color:rgb(255, 255, 0)">actual changes </span>in each commit
```bash
git log --online --patch
```
