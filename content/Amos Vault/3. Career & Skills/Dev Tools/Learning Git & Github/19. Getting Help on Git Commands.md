i. You can visit the git documentation.
- Google the command eg: <span style="color:rgb(255, 148, 148)">git.config</span> open the page and view all it options.

ii. On the terminal
- Type the command and the --help
ie:
```bash
git config --help
```

iii. Getting a short summary about the command.
- Use -h
ie:
```bash
git config -h
```
