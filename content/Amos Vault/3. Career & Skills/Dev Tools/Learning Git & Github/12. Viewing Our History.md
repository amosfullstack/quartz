## Overview

QN : We have made some commits so far, but  where are these commits.
<span style="color:rgb(0, 176, 80)">Answer</span> :  We use the git log command to look at our history.
ie: 
```bash
git log
```
Result : The result is our history of the <span style="color:rgb(0, 176, 80)">commits </span>we commited, sorted from the latest to the <span style="color:rgb(0, 176, 80)">earliest</span> .

#### Explanation :
i. Each commit has a <span style="color:rgb(255, 255, 0)">unique identifier </span>. This is the hexadecimal string that git autmatically generates.

ii. HEAD->Master : HEAD is the <span style="color:rgb(255, 255, 0)">reference to the branch</span> that we are currently working on. So, HEAD->master, head pointing to master says that we are currently working on the master branch.

iii. For each commit, we can see the <span style="color:rgb(255, 255, 0)">author</span>, <span style="color:rgb(255, 255, 0)">email</span>,<span style="color:rgb(255, 255, 0)">date</span> and <span style="color:rgb(255, 255, 0)">time</span> the commit was created.Also, the <span style="color:rgb(255, 255, 0)">description</span> of the commit.

iv. If commits are on multiple pages. Press <span style="color:rgb(255, 255, 0)">space</span> to go to the next Page. Press q to <span style="color:rgb(255, 255, 0)">quit</span> .

Log Command Options :

a. To see short summary of the commits
```bash
git log --oneline
```

b. Revering the sort order of the commits history
```bash
git log --oneline --reverse
```
