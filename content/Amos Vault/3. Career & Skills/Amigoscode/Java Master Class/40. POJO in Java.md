POJO : Stands for Plane Old Java Object

Basically POJO Means:
- This Emphasizes that a given Object is a ordinary java Object.
- The Object has no association with any Framework.
- Example a Class <span style="color:rgb(255, 255, 0)">Person</span> is a POJO.
- Becase : It has no association with any Object. 
-              : It does not Extend any Class.
-              : Nor it does not implement any Class.


- A POJO Focuses on the business Logic.
- It has the information of the business Logic.

- A POJO Class can be used within any Framework.
