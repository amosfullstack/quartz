What is Abstraction?


What is the Abstract Key word?
- The abstract key word is used with classes that its objects can <span style="color:rgb(255, 148, 148)">not be instantiated</span>.

Example:
- If you have the animal class
- And you have a Cat subclass
- And a dog subclass.
- There is <span style="color:rgb(255, 148, 148)">no point </span>of instantiating the Animal Object.
- Therefore the Animal Super-Class should be an <span style="color:rgb(255, 0, 255)">abstract class</span> and it should start with the abstract keyword.


###### i. Abstract keyword on classes.

ie:
```java
public abstract class Animal{
  //The Superclass in which you can not instantiate an object.
}
```

Instantiating the objects.
```java
Dog dog1 = new Dog();

//Also this works
Animal dog1 = new Dog(); //Because a dog is an animal.
```


ii. Abstract Methods.
- If the super class has an abstract method. 
- It becomes <span style="color:rgb(255, 0, 255)">compulsory</span> for Every subclass to <span style="color:rgb(255, 0, 255)">override</span>/or define its own implementationof that method.
- The abstract method must Exist in all subclasses.

NB : Abstract methods should only <span style="color:rgb(0, 176, 80)">Exist</span> only within <span style="color:rgb(255, 255, 0)">Abstract Classes </span>or <span style="color:rgb(255, 255, 0)">Interfaces</span>.

Example:
- Abstract method in super-class
```java
public abstract class Animal{
   public abstract void makeSound()
}
```

- In the subclass
```java
public class Cat{

@Override
public void makeSound(){ //Forced to override, it is abstract in super Class
   sout("Miaoo");
}

}
```
