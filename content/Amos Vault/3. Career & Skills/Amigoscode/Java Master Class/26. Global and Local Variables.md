What is a <span style="color:rgb(255, 148, 148)">global Variable</span>?
- A global variable is a variable that is<span style="color:rgb(255, 255, 0)"> accessed </span>by anything within the same class or outside of the class.
- So, global variables can be accessed by <span style="color:rgb(255, 255, 0)">all functions </span>within the class.
- The are declared inside the class but outside the methods.
- A global variable are a scope for anything within the class.


What is a <span style="color:rgb(255, 148, 148)">local Variable</span>?
- The local variable is only visible within the scope of the function.
- The local variables can not be accessed by other methods in the same class.
