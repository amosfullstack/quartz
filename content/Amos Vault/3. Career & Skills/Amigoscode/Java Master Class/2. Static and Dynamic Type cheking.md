What is <span style="color:rgb(255, 0, 255)">type(data types)</span>?
- Is the classification identifying one of various types of Data. Eg(Strings, intergers,Objects, Arrays, Maps etc..

What is <span style="color:rgb(255, 0, 255)">Type Checking </span>?
- Is the process of  identifying and enforcing constraints of types.
- This may occur either at complle time(<span style="color:rgb(255, 255, 0)">static type cheking</span>).
- Or it may occur at Run time (<span style="color:rgb(255, 255, 0)">Dynamic type checking</span>).

So, Java is Static checked while Javascript is dynamically checked.

<span style="color:rgb(255, 148, 148)">Static Type checking</span> :
- A language is statically-typed if the type of a variable is known at compile time instead of at run time.
- This allows many type errors to be caught early in the development cycle.

<span style="color:rgb(255, 148, 148)">Dynamic Type checking </span>:
- Is the process of veryfing the type safety of the programme at runtime.
- Commony dynamically typed languges include. (Groovy, Javascript, Lisp, Lua, PHP, Objective-C, Prolog, Python, Ruby, Smalltalk and Tcl)

In Java:
```java
int number = "Amos"; 
//This wont work and will bring compilation error.
```

In Javascript:
```javascript;
let number = "Amos";
//This will work without any issue, Becase type safety is verified at runtime.
```
