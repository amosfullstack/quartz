
___

- A programmer can create custom both checked and unchecked Exceptions.
- This can be done by creating a class for the exception.
- Right click,Generate,select overide methods, pick Exception with String..

Practicals :
Creating checked and unchecked Exceptions and then using them in the main method.

i. Unchecked Exception
```java
public class MyUnCheckedException extends RuntimeException{  
  
    //Allows the ability to pass a message into our Exception  
    public MyUnCheckedException(String message) {  
        super(message);  
    }  
}
```

ii. Checked Exception
```java
public class MyCheckedException extends Exception{  
  
    //Allows the ability to pass a message into our Exception  
    public MyCheckedException(String message) {  
        super(message);  
    }
```


Using the Exceptions
i. For unchecked
```java
//Use the exceptions  
public static int divide(int a,int b){  
    if(b==0){  
        throw new MyUnCheckedException("Can not divide by 0!");  
    }  
    return a/b;  
}
```

ii. For Checked
- 
```java
//Use the exceptions  
public static int divide(int a,int b) throws Exception{  
    if(b==0){  
        throw new MyCheckedException("Can not divide by 0!");  
    }  
    return a/b;  
}
```



GENERALLY.
i. Checked Exceptions are created by extending the Exception Class.
ii. The Unchecked Exceptions are created by Extending the RuntimeException Class.
