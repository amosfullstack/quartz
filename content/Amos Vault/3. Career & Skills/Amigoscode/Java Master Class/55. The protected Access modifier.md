What is a protected keyword?
- The attributes that have a protected access modifier will be accessed by the <span style="color:rgb(255, 255, 0)">super class</span> and its <span style="color:rgb(255, 255, 0)">subclasses</span> 
- If you user <span style="color:rgb(255, 148, 148)">private</span> to the superclass attributes, then the subclasses will <span style="color:rgb(255, 148, 148)">not be able</span> to access them.

ie:
- Super Class
```java
public class Employee{
 protected String name;
}
```

- Sub Class
```java
public class Programmer extends Employee{
  sout(this.name); //The superclass name is accessed to the subclass because of the protected keyword
}
```
