What is Method Overriding?
- This reffers to changing the implementation of the method in the Sub-Class that has already been implemented in the Super-Class.
- The methods have the same name but different implementation.

ie:
Method in the Super-Class
```java
public void SayHi(){
  sout("Hello Amos);
}
```

Method in the Sub-Class
```java
public void sayHi(){
 sout("Hi");
}
```

- These these two different methods, have the same <span style="color:rgb(255, 255, 0)">Signature</span> but Perform different <span style="color:rgb(255, 255, 0)">Behaviour</span> .
