What are arithmetic operators?
- Arithmetic operators are characters that enable perfoming of mathematical operations in java.

i. Addition(+)
ii. Substraction(-)
iii. Multiplication(`*`)
iv. Division(/)
v. Modulus(%) : Returns the remainder.

ie: 
```java
int number1 = 10;
int number2 = 3;

sout(number1+number2);
sout(number1-number2);
sout(number1*number2);
sout(number1/number2);
sout(number1%number2);
```

vi. Increment operator
```java
int number = 0;
sout(number++);
```
> The above code says, return a value then increment it by one.
> So, it prnts zero first before incrementing it.

To incremet the number first, out the increment operator before.
ie: 
```java
int number = 0;
sout(++number);
```
> Increment first, then print the number.
> So, 1 will be printed.As 0 is incremented first by 1.
