#### 2. while loop
- Is the kind of loop that repeats a block of code as long as the condition is true.
- It stops only when the condition turns false
ie:
```java 
while(true){
  sout("Hello");
}
```

ie:
```java
boolean keepLooping = true;
while(keepLooping){
 sout("Hello);
 keepLooping = false; //Turn the condition to false
}
```

#### 3. do while loop
- This loop executes the block of code atleast once.
- Then it continues looping as long as the given condition is true.
- It stops when the condition turns false.
ie:
```java
boolen keepLooping = false

do{
  sout("Hello");
}while(keepLooping)
```
- Though the condition is false, the loop will first execute once.
