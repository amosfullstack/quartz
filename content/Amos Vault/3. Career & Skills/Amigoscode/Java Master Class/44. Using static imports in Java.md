- The static keyword can be used to import some packages hence improve code readability.

ie: Instead of using
```java
System.out.println(Math.PI);
System.out.println(Math.max(12,20));  
System.out.println(Math.min(10,5));
```

use : Static import.
ie:
```java
import static java.lang.Math.max;  
import static java.lang.Math.min;

//Inside main
System.out.println(max(12,20));   //No need of starting with Math.
System.out.println(min(10,5));

```
