## Overview
- With the static key word, we can use the static block Initialization.
- Static block initialization is used to <span style="color:rgb(255, 255, 0)">execute code</span> that <span style="color:rgb(255, 255, 0)">initializes</span> the value of the static field.


Static Initialization:
- Any piece of code can be executed, inside the static block.
- There should be no return statement inside the static block.
- There are also no arguments.
- And this piece of code is executed by the JVM, when the class is called.
- You can have multiple static initializer blocks in a Class.
```java
static int count;  
  
static {  
    System.out.println("Start: static initialization");  
    //Execute any piece of code here  
    //Eg    count = 0; //The Static field has been initialized to zero.  
    System.out.println("End: static initialization");  
}
```

Uses Examples(Where static block initialization can be used);
- Establishing connection to the database, perfom a query and then assign the value to any static field we have.
- If we do not have a static field declared before, the static block initialization can not be done.
- We are only able to work with static fields, inside the static initialization block.
