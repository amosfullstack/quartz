What is an Array?
- An array is a data structure that allows storage of multiple values inside one variable.

Creating an array in Java :
datatype[] name = new datatype`[specifysizeofArray]`
ie:
```java
int[] numbers = new int[5];

//Adding values into an Array
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
```

- To print the values in the array use the <span style="color:rgb(255, 255, 0)">Arrays.toString()</span> method:
- This will <span style="color:rgb(255, 0, 255)">import java.util.Arrays </span>package, so that you can invoke the method.
ie:
```java
System.out.println(Arrays.toString(numbers));
```

- Dont pass the array itsel without the method, it will print Random hexadecimals.
ie:
```java
sout(numbers);
```

NB: Once we define the size of array, we can not insert the values that are out of bound.
> If you for it, the exception array out of bound will be thrown.

Some Array Methods.
i. <span style="color:rgb(255, 255, 0)">.length</span> : Returns the length of the Array.
ie: 
```java
sout(numbers.length);
```


###### Short hand version of Array creation

###### Arrays Null Values
- When the array is delcared without inserting values, Its default values are <span style="color:rgb(255, 255, 0)">zeros</span>.
ie:
```java
int[] numbers = new int[3];
sout(Arrays.toString(numbers));
[0,0,0]  //Will be outputed.
```

- But you can set the default values of an array other than zero using <span style="color:rgb(255, 255, 0)">Arrays.fill()</span> method.
ie:
```java
int[] numbers = new int[3];
Arrays.fill(numbers,-1);  //Setting the default values to be -1
```
NB: This is for primitives.

What about for reference types?
- The default values for the arrays in the refference types is <span style="color:rgb(255, 255, 0)">null</span>.
ie:
```java
String[] names = new String[3];
sout(Arrays.toString(names));
[null,null,null] //Will be outputed
```

- You can create the default values also, for the reference types.
```java
Arrays.fill(names,"Jamila");
```
