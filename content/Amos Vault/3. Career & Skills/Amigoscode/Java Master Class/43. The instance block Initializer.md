What is the instance block initializer?
- The instance block initializer is just the same as the static block initializer.
- It is just curly braces without any keyword.
ie:
```java
{

}
```

- Inside the Instance Block initializer, you can refer to any propert that you have in a class and assign default values.
eg: Not practical but just an example
```java
String firstName;  
int age;  
  
{  
    this.firstName = "Amos";  
    this.age = 24;  
}
```

- You can also  execute code inside the Instance block initializer, that can be used to assign values to the fields.
ie:
```java
String firstName;

{
//Code to be executed that perfome action on the values
}
```
