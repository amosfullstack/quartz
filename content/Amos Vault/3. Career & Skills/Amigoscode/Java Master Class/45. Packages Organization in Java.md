- <span style="color:rgb(255, 255, 0)">Packages</span> is just a term in Java, in other languages its just <span style="color:rgb(255, 255, 0)">folder</span>.
- Do not always leave thing in the root Package,separate Classes into separate packages.
- <span style="color:rgb(255, 255, 0)">Related Classes</span> should be in the Same package.

ie: 
<span style="color:rgb(255, 255, 0)">com.amos </span>(Root Package)
    <span style="color:rgb(255, 148, 148)">com.amos.Car</span> (Car Packege)
      ->Car.java (Contains the Car Class)
    <span style="color:rgb(255, 148, 148)">com.amos.Cat</span> (Cat Package)
      ->Cat.java (Contains the Cat Class)
    <span style="color:rgb(255, 148, 148)">com.amos.Person</span> (Person Package)
      ->Person.java (Contains the Person Class)
      ->Gender.java (Contans the Gender enum)
