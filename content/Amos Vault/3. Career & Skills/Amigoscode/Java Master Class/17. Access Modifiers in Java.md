What is access Modifiers?
- Basically, access modifier reffers to access level.
- How classes, methods, attributes and constructors can be accessed.

Access Levels include :
- <span style="color:rgb(255, 0, 255)">default</span>, <span style="color:rgb(255, 0, 255)">public</span>, <span style="color:rgb(255, 0, 255)">private</span> and <span style="color:rgb(255, 0, 255)">protected</span>.


i. public
- <span style="color:rgb(255, 255, 0)">public</span> <span style="color:rgb(255, 255, 0)">access</span> modifier means that the class can be <span style="color:rgb(255, 0, 255)">accessible by any other class</span>.
- Same applies to the method, the public method is accessible by any other class. Inside the package and outside the package.
- The public acess modifier can be used with a local variable
- It is used just with <span style="color:rgb(255, 255, 0)">Classes</span>, <span style="color:rgb(255, 255, 0)">methods</span>, <span style="color:rgb(255, 255, 0)">constructors</span>, and Class Level variables .


ii. private
- <span style="color:rgb(255, 255, 0)">private</span> access modifier means that the attribute can be accessed on in the scope of the same Class.

iii. default 
- <span style="color:rgb(255, 255, 0)">default</span> access modifier means that the method/attribute/class can be accessed by anything inside the same package.
- declaring default means using neither private, nor public.
ie:
```java
static String userName = "Amos"; //This can be accessed by anything inside the same pacakge
```
