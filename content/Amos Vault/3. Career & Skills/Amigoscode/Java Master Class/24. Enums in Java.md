What is Enum?
- <span style="color:rgb(255, 0, 255)">enum</span> is the keyword in Java that allows us also to define constants.
- Declaration should be outside of the main method.

Example:
- Defining the constants for gender .
```java
enum Gender {
   MALE,
   FEMALE
}

//Using the constant in the main method
sout(Gender.MALE);
sout(Gender.FEMALE);


//Here The male and female have been given a datatype called gender that we can work with.
```

- Defining constants for Tshirt Sizes.
```java
enum TshirtSizes {
  S,
  M,
  L,
  XL
}

```


The importance of enums constants over normal constants, is that we now can have the datatype that we can work with. In our case <span style="color:rgb(255, 255, 0)">Gender</span> and <span style="color:rgb(255, 255, 0)">TshirtSize</span> .
Compare with normal:
```java
public static final String MALE = MALE;
public static final String FEMALE = FEMALE; //This is somwhow awkward
//Here you do not have a type, because they are just Strings

```
