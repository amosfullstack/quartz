What are varibales?
- Variables are containers for storing values. Are the location in the memory where the values used in the programme are stored.

#### Primitives
- Primitives are variables that store simple values. Example: integer, string

i. int
- <span style="color:rgb(255, 255, 0)">int</span> is a data type that stores whole numbers.
- int has a size of 32 bits.
- The numbers range from <span style="color:rgb(255, 0, 255)">Minimum -2147483648 to Maximim  2147483647</span>.
```java
int number1 = 10;
int number2 = 10;
int result = number1 + number2;
```

ii. double
- <span style="color:rgb(255, 255, 0)">double</span>  is a datatype that stores decimal values
```java
double pi = 3.14;
```

iii. boolean
- <span style="color:rgb(255, 255, 0)">boolean </span>is a datatype that stores true or false value
```java
boolean isValid = true;
boolean isAdult = false;
boolean isOnline = true;
```

iv. char
- Is a datatype that stores character values.
```java
char grade = 'A';
char bloodGroup = 'O';
```

v. short
- <span style="color:rgb(255, 255, 0)">short</span> is also used to store the number values.
- It has a size of  <span style="color:rgb(255, 0, 255)">16 bits</span> .
- The numbers should range from <span style="color:rgb(255, 0, 255)">Minimum -32,768 to 32,767 Maximum</span>.
ie:
```java
short x = 30000; //Allowed
short y = 40000; //Not allowed, it will bring an error.
```

vi. long
- <span style="color:rgb(255, 255, 0)">long</span> is used to store very large number values.
- It has a size of <span style="color:rgb(255, 0, 255)">64 bits</span>.
- It ranges from  <span style="color:rgb(255, 0, 255)">Minimum -92233720368547758008 to Maximum 92233720368547758008</span>

vii. byte
- <span style="color:rgb(255, 255, 0)">byte</span> is used to store the number values.
- It has a size of <span style="color:rgb(255, 0, 255)">8 bits</span>.
- The numbers should range from <span style="color:rgb(255, 0, 255)">Minimum - 128</span> to <span style="color:rgb(255, 0, 255)">+127 Maximum</span>.
ie:
```java
byte n = 127; //Correct
byte x = 128; // This is incorrect it does not work.
```

viii. float
- <span style="color:rgb(255, 255, 0)">float</span> is a datatype that stores decimal values. It allows to store larger values than double, therefore taking large space in memory.


You can add underscore for better readability of numbers. The underscore do not affect the number or add any value, it is just like a comma in normal life.
```java
int amount = 1000000008; //Not well readable
int amount  = 1_000_000_008; // Very easly readable, its 1 billion
```
