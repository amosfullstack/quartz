What are DAO Classes?
- DAO : Stands for Data Access Object.
- These are Classes responsible for interacting with <span style="color:rgb(255, 255, 0)">Databases</span> or <span style="color:rgb(255, 255, 0)">Stores</span> for your Application.
- DAO Classes are responsible for handling how <span style="color:rgb(255, 255, 0)">Data</span> is stored.

Example :
- You want to Store Cars to the Database.
- You want to store cats to the Database.
- You want to Store Cars that are going to the garage to the Database for Example.
- A database could be In memory <span style="color:rgb(255, 255, 0)">Array</span>, a <span style="color:rgb(255, 255, 0)">File</span> or a <span style="color:rgb(255, 255, 0)">Real Database</span>.

NB : The Business Logic and Checks should be handled inside the Service Class, before communicating to the DAO Class.
