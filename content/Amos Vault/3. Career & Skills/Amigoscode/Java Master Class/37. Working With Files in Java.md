## Overview
i. In Java We have ability to:

- Delete a File.
- Read from a File.
- Write to a File etc.. 

ii. There are many ways to work with files in Java, People use different Classes and different libraries.
iii. We are going to learn one of the many ways to work with files in Java.


Practicals.
i. Creating A file in Java.
- We are using a <span style="color:rgb(255, 255, 0)">File</span> Class. it is found in <span style="color:rgb(255, 255, 0)">java.io</span> package.
ie:
```java
try {  
    File file = new File("src/foo.txt");  
    file.createNewFile();  
} catch (IOException e) {  
    throw new RuntimeException(e);  
}
```
- It is compulsory to handle the checked Exception IOException before compling the code.

ii. Deleting a file
```java
file.delete();

//ie
if(file.exists()){
   file.delete();
}
```

iii. Create a method that when invoked, it creates a text file in the src folder.
```java
  
//Method that Creates a file  
public static File createFile(String path){  
    File file = new File(path);  
    try {  
        if (!file.exists()) {  
            file.createNewFile();  
        }  
    } catch (IOException e) {  
        System.out.println(e.getMessage());  
    }  
  
    return file;  
}
```
- The File Class has been used.
- The Checked Exception is handled

iii. Write inside the File you created above.
```java
//Writing to the file  
try {  
    FileWriter fileWriter = new FileWriter(file);  
    PrintWriter writer = new PrintWriter(fileWriter);  
    writer.println("Hello");  
    writer.println("World");  
  
    writer.flush(); //Flush the contents into the file  
    writer.close(); //Close the writer  
} catch (IOException e) {  
    System.out.println(e.getMessage());  
}
```
- Writing to the file we user the <span style="color:rgb(255, 255, 0)">FileWriter</span> Class.
- The file Writer is also passed into the <span style="color:rgb(255, 255, 0)">PrintWriter</span> Class.
- The print writer, writes to the file,flush and then close.
- The above code,overites the content of the file, if you write new Data.

: If you want to append to a file, add the parameter true..
```java
FileWriter fileWriter = new FileWriter(file,true);
//append:true
```

iv. Refactor a writing to file code into a method that accepts a file and append boolean.
```java
  
//Method to Write to a File  
private static void writeToFile(File file,boolean append){  
  
    //Writing to the file  
    try {  
        FileWriter fileWriter = new FileWriter(file,append);  
        PrintWriter writer = new PrintWriter(fileWriter);  
        writer.println("Hello");  
        writer.println("World");  
        writer.println("My");  
        writer.println("Name");  
        writer.println("Is");  
        writer.println("Amos");  
        writer.println("Juma");  
  
        writer.flush(); //Flush the contents into the file  
        writer.close(); //Close the writer  
    } catch (IOException e) {  
        System.out.println(e.getMessage());  
    }  
}
```


v. Reading from the file.
- Here we use the <span style="color:rgb(255, 255, 0)">Scanner</span> Class. But the scanner class accepts the the file as an argument. or the <span style="color:rgb(255, 255, 0)">file source</span>.
- The is a FileNotFoundException that should be handled.
ie:
```java
//Reading from a file  
try{  
    Scanner scanner = new Scanner(file);  
    while (scanner.hasNext()){  //Check if the file has next line
        System.out.println(scanner.nextLine());  
    }  
  
} catch (FileNotFoundException e) {  
    System.out.println(e.getMessage());  
}
```


v. The Writing to file code can be refactored such that, we do not have to fush and Close everytime.
- This can be done using try with resources.
ie:
```java
//Method to Write to a File  
private static void writeToFile(File file,boolean append){  
  
    //Writing to the file  
    try (  
        FileWriter fileWriter = new FileWriter(file, append);  
        PrintWriter writer = new PrintWriter(fileWriter)  
    ){  
        writer.println("Hello");  
    }  
    catch (IOException e) {  
        System.out.println(e.getMessage());  
    }  
}
```
