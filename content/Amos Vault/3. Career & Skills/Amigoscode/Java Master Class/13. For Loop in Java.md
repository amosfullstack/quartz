What is a Loops?
- A loop is construct tha allows us to repeat a block of code.

#### 1. for loop
- Repeats the block of code for a given number of times.
```java
for(int i=0; i<=10; i++){
sout("Hello");
}
//This will repeat printing hello, Eleven times. Means from 0 to 10 repetition
```

i. Looping through Arrays with for loop.
```java
String[] names = {
    "Amos", //<- name used in enhanced for loop
    "Juma",
    "Salehe",
    "Kim"
};

for(int i=0; i<names.length; i++){
 sout(names[i]);
}
```

ii. But we can loop the array using the <span style="color:rgb(255, 255, 0)">enhanced</span> for loop
```java
for(String name : names){ //name stands for each name in the array names
  sout(name);
}
//This works perfectly file.
//It eliminates a lot of moving parts from the normal for loop
```
- This loop knows how to move to the next element, and it also knows when to stop.

<span style="color:rgb(0, 176, 80)">The Difference between the two loops</span> : So as to know when to use which.

a. <span style="color:rgb(255, 255, 0)">Normal</span> for loop is used when you want to have access to the index.
- So, it is used when you want to access the elements in the array by the given index.
ie:
```java
for(int i=0; i<names.length; i++){
	String prev = names[i-1];
	String current = names[i];
	String next = names[i+1];
}
```

b. For <span style="color:rgb(255, 255, 0)">Enhanced</span> for loop, you do not have control. It just loops from start to finish. You do not have access to the indexes.
- You can not get to the previous or next.
