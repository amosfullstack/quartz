What is java Beans?
- Is a class that has to Obey three Contacts so that other frameworks can work with it.

Three Contracts that a java bean should have:
i. The class should hava a <span style="color:rgb(255, 255, 0)">no arg constructor</span>.
- The class should have the constructor without argumenents
ie: 
```java
public Car(){
}
```

ii. All the Attributes (the <span style="color:rgb(255, 255, 0)">fields</span>) should be <span style="color:rgb(255, 255, 0)">private</span>.
ie:
```java
private String regNumber;
private BigDecimal price;
```

iii. The Class should implement <span style="color:rgb(255, 255, 0)">Serializable</span> 
ie:
```java
public class Car implements Serializable{
}
```

A class that implements Serializable, means that its Object can be written to Streams eg Files,Databases etc..

NB : A Bean is not a <span style="color:rgb(255, 255, 0)">POJO</span>, as we have said that a pojo can not implement anything .


Therefore:
- In Sping Boot, java bean is mostly used because there are classes that are dependent on the framework.
