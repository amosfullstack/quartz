- It can alse be used for frontend but not recommended, there are many frameworks such as angular, react etc.

N-Tier Architecture.
<span style="color:rgb(255, 148, 148)">Clients</span> : Send Request to the backend.
<span style="color:rgb(255, 148, 148)">Api Layer</span> : Java takes the request from the clients and submits it to the backend.
<span style="color:rgb(255, 148, 148)">Business Layer</span>: Where the business Logic lies. ie Where you make your business decisions.
<span style="color:rgb(255, 148, 148)">DAO Layer</span> : Where you can store data against any database of choice.ie (Mysql database,a file,  an nosql database etc..)

Java:
- Java is the compiled/interpreted language.
- Java script is the interpreted Language.

<span style="color:rgb(255, 255, 0)">Compiler</span> : The compiler turn code statements into machine cod(0101) that can be understood by the computer's processor.

<span style="color:rgb(255, 255, 0)">Interpreter</span> : The interpreter reads each statement of code then converts it or executes it directly. So interpreter takes the code and executes it line by line.


| Compled Language                                                 | Interpreted Languge                  |
| ---------------------------------------------------------------- | ------------------------------------ |
| i. Code can be executed directly by the computer's CPU.          | Another Programme executes the code. |
| ii. Code must be transformed into machine readable instructions. | No transformation needed.            |
| iii. Runs faster.                                                | Runs slower.                         |
| iv. Better for perfomance.                                       | Faster development.                  |
| v. C++, GoLang                                                   | Python,Javascript                    |
Java sits in Between:
<span style="color:rgb(255, 255, 0)">Because</span> : Java has something JVM, so the java source code is first compiled into something called bytecode, then the JVM takes the bytecode and them executes it.
