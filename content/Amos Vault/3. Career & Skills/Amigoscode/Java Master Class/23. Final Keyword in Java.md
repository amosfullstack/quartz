What is the final keyword?
- The final keyword allows you to <span style="color:rgb(255, 255, 0)">create constants</span> .
- Also <span style="color:rgb(255, 255, 0)">prevents inheritance</span> .
- Also prevents <span style="color:rgb(255, 255, 0)">method overiding</span> .

i. Creating constants
```java
final var userName = "Amos";
userName = "Juma"; //This wont works because its a constant
```
- Creating contants enables you to create variables that users are not able to modify.

- Nice convention for creating constants
```java
public static final String NAME = "Amos";
```

- Having the final keyword on the properties of your classes is really imporant.

NB : We shall cover later about the final keyword on inheritance and method overiding.
