What is Inheritance?
- Is the property where by the <span style="color:rgb(255, 255, 0)">subclass</span> <span style="color:rgb(255, 255, 0)">inherits</span> all the properties and methods of the subclass.
- Inheritance Allows us to create the <span style="color:rgb(255, 255, 0)">new Class</span> from the <span style="color:rgb(255, 255, 0)">existing class</span>.
- Inheritance helps to eliminate code <span style="color:rgb(255, 255, 0)">duplicaiton</span>

i. Implementation of the Super Class
```java
public class Employee {  
    private String name;  
    private int age;  
    private String address;  
    private String experience;  
  
    public Employee(String name, int age, String address, String experience) {  
        this.name = name;  
        this.age = age;  
        this.address = address;  
        this.experience = experience;  
    }  
}
```

ii. Implementation of the programmer Subclass
- Use the <span style="color:rgb(255, 0, 255)">extends</span> keyword to create subclass from the supers class
```java
public class Programmer extends Employee{  
  
  //Programmer attributes
    private String[] programmingLanguages;  
  
  //Constructor
    public Programmer(String name, int age, String address, String experience) { 
        super(name, age, address, experience);  
    }  
}
```


iii. Implementing the Manager Subclass
```java
public class Manager extends Employee{  
  //Manager self attribute
    private int TeamSize;  
  
  //Constructor matching the super class
    public Manager(String name, int age, String address, String experience) {  
        super(name, age, address, experience);  
    }  
}
```


iv. Creating the Constructor for the progammer subclass that includes programming Languages attribute
```java
public Programmer(String name, int age, String address, String experience, String[] programmingLanguages) {  
    super(name, age, address, experience);  //Super class constructor is always included
    this.programmingLanguages = programmingLanguages;  
}
```


The <span style="color:rgb(255, 0, 255)">super</span> keyword in the constructor.
- The super keyword refers to the constructor of the parent class.
- Thus, the constructor of the super class , will be automatically present in the subclass.

v. Calling the methods that are in the superclass, to be used in the subclass
```java
super.sayHi(); //This will call the sayHi() method in the super class
```
