## Overview
- The Scanner Class allows us to Take user input from the Console.
- We Shall see how the Scanner Class Reads files as well.

###### The Scanner Class
- The <span style="color:rgb(255, 255, 0)">Scanner</span> class  is used to take the <span style="color:rgb(255, 255, 0)">user input</span> .
- It is found in the <span style="color:rgb(255, 255, 0)">java.util</span> package .
ie:
```java
Scanner scanner = new Scanner(System.in);
```
From the code above:
>- <span style="color:rgb(255, 148, 148)">scanner</span> : The Scanner Object has been instantiated .
>- <span style="color:rgb(255, 148, 148)">System.in</span> : Allows us to take input from the console .
>- NB : Different parameters appart form System.in can be passed in the Scanner object,but in our case we are interested with console user input.

- Scanner methods Available.
```java
.nextInt();
.nextLine(); //Takes any given string from the console and stores it in the variable
.next();
.close();
.delimeter();
.skip();
```



#### Practicals :
i. Creating the Scanner to take the String username from the user.
ie :
```java
System.out.println("What is you name?");  
//Instantiating the Scanner Object  
Scanner scanner = new Scanner(System.in);  
  
//Storing the Result from the console  
String userName = scanner.nextLine();  
  
//Outout the result  
System.out.println("Hello "+userName);
```

ii. Limiting the Data Type to take on interget datatype.
```java
System.out.println("What is your Age?");  
int age = scanner.nextInt();  
System.out.println("Your age is "+age+" years!");
```
- Any input other than an interger will throw an error.
