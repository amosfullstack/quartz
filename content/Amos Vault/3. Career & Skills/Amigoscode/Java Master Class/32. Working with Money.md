## Overview
- When working with money, Such as printing prices. Do not work with <span style="color:rgb(255, 148, 148)">double</span>.
- This is becasue there is a problem with <span style="color:rgb(255, 148, 148)">precision</span>.
Example:
```java
double price1 = 0.03;  
double price2 = 0.02;  
double price = price1-price2;  
System.out.println(price);

//RESULT
0.009999999999999998

//EXPECTED 
0.01
```
> So because of presicion we do not use double when Working with money.


#### Working with money
QN : What Data Type should we use when working with Money?
<span style="color:rgb(0, 176, 80)">ANSWER</span> : When working with money she Datataype we should use is <span style="color:rgb(255, 255, 0)">BigDecimal</span> Class.
ie:
```java

BigDecimal price1 = new BigDecimal("0.03");  
BigDecimal price2 = new BigDecimal("0.02");  
BigDecimal price = price1.subtract(price2);   //Methods available for BigDecimal Object
System.out.println(price);

//RESULT
0.01

//EXPECTED
0.01
```

ii. The methods in Big Decimal .add and constants
```java
    //There are ZERO,ONE,TWO  
BigDecimal number1 = BigDecimal.TEN; //There is an array of Constants 0 to 10  
        //From three to nine use valueOf  
BigDecimal number2 = BigDecimal.valueOf(6);  
BigDecimal result = number1.add(number2);  
System.out.println(result);
```

iii. Other BigDecimal methods include
```java
.max(); //Returns the maximum among numbers
.compareTo(); //Returns the difference of the two  numbers
```
