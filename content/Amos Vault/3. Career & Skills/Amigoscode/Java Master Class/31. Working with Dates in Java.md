## Overview
- Previously Java had a terrible API when working with dates.
- But now there are many improvements, so there a couple of classes that we can use to work with dates.

Practicals

i. Using the <span style="color:rgb(255, 255, 0)">LocalDateTime</span> class.
- This comes form the packege <span style="color:rgb(255, 255, 0)">java.time</span>
- This class contains many methods such as <span style="color:rgb(255, 255, 0)">now</span>, <span style="color:rgb(255, 255, 0)">from</span>, etc..
- It gives the current local date and time from your machine
ie:
```java

import java.time.LocalDateTime;
 
LocalDateTime myTimeNow = LocalDateTime.now();
sout(myTimeNow); //Prints my current Date Time and Milliseconds


//Other methods in this class
System.out.println(myTimeNow.getMonth());  
System.out.println(myTimeNow.getDayOfYear());  
System.out.println(myTimeNow.getDayOfWeek());  
System.out.println(myTimeNow.getHour());
```

ii. Using the <span style="color:rgb(255, 255, 0)">LocalDate</span> class.
- Allows you to work with Local Dates Only Eliminating Time.
- All the methods concerning date from above work, Except those concerning Time.
```java
import java.time.LocalDate;

LocalDate myDateNow = LocalDate.now();  
System.out.println(myDateNow );  
  
  //Methods that may be used
System.out.println(myDateNow.getMonth());  
System.out.println(myDateNow.getDayOfYear());  
System.out.println(myDateNow.getDayOfWeek());                        
```


iii. Using the <span style="color:rgb(255, 255, 0)">LocalTime</span> class.
- Allows you to Work with Local time only
```java
import java.time.LocalTime;

LocalTime mylocalTime = LocalTime.now();  
  
//Other methods to Work with  
System.out.println(mylocalTime.getHour());  
System.out.println(mylocalTime.getMinute());  
System.out.println(mylocalTime.getSecond());
```

- Creating a specific Date(you pass year,moth,date,hour,minute,second etc..)

```java
LocalDateTime myBirthDate =  LocalDateTime.of(2001,03,21,14,53);  
  
System.out.println(myBirthDate);

//This works the Same for LocalDate and LocalTime
```

vi. To know the Local Time and Date of A particular Zone.
- This involves combining the <span style="color:rgb(255, 255, 0)">LocalDateTime</span> class and the <span style="color:rgb(255, 255, 0)">ZoneId</span> Class.
- The zoneId is passed as the Paramenter of LocalDateTime.now() method.
ie:
```java
LocalDateTime nowInPanama = LocalDateTime.now(ZoneId.of("America/Panama"));  
System.out.println(nowInPanama);
```

vii. Using the <span style="color:rgb(255, 255, 0)">ZoneDateTime</span> class.
- This allows you too see the Date , time and also the actual Zone.
ie:
```java
ZonedDateTime currentTimenZone = ZonedDateTime.now();  
System.out.println(currentTimenZone);
```
