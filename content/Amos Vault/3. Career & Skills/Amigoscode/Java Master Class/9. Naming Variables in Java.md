- The best way to name variables in java. Is to always to use names that have meaning.
Eg: instead of <span style="color:rgb(255, 148, 148)">goals</span> say <span style="color:rgb(0, 176, 80)">numberOfGoals</span> 

i. Use Camel Case convention.
- The first word is lover case, then words that follow start with upercase.
ie:
```java
int numOfGoals = 10;
```

ii. For Boolean use is and has.
- Always ask a question when declaring a boolean.
ie:
```java
boolean isAdult = true;
boolean hasPhoneNumber = false;
```

iii. For Methods and Classes, use camel case convetion.
