i. <span style="color:rgb(255, 255, 0)">break</span> : The break key work is used to exit the loop.
              : It was used with the switch statement so exit if the condition was met.
ie:
```java
var numbers = new int[]{1,2,3,4,5};
for(number : numbers){
  if(number==3){
    break;
  }
  sout(number)
}
```
> From the code, when the loop reaches number three, it will stop execution.


ii. <span style="color:rgb(255, 255, 0)">continue</span> : The continue keyword is used to skip the value in the loop
                   : It ends the current execution of the loop, then jumps to continue with the next itteration.
   ie :
   ```java
   var numbers = new int[]{1,2,3,4,5};
for(number : numbers){
  if(number==3){
    continue;
  }
  sout(number)
}
   ```

> From the code, number 3 will be skipped. It will not be printed.


iii. <span style="color:rgb(255, 255, 0)">return</span> : The return keyword stops the entire main method.
ie :
```java
var numbers = new int[]{1,2,3,4,5};
for(number : numbers){
  if(number==3){
    return;
  }
  sout(number)
}
```
> This will stop the entire program, thus nothing below the return keyword will execute.
