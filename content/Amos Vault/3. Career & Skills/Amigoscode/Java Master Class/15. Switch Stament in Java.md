- The switch statement can be used to substitute the if else statement.
- So, the switch statement can be used to replace nested if conditions.

Example :
```java
char grade  = 'B';

switch(grade){
  case 'A':
     sout("Excellent);
     break;
  case 'B' :
      sout("Very Good");
      break;
   default :
      sout("Fail"); 
      //For the default there is not need for break      
}
```


- The datatype that can be passed in the switch statement at the condition to check are:
    ie:   byte, short, int, char, enums, String

- Also <span style="color:rgb(255, 255, 0)">Wrapper Classes</span> can be passed.
- ie:  Byte, Short, Interger, Character

#### Storing the result of the switch Statement into a Variable

- Declare a variable and assign it to the switch statement
ie:
```java
String result = switch(grade){
                case 'A' -> "Excellent"'
                case 'B' -> "Pass";
                case 'C' ->"Tried";
                default  ->"Failed";
};
```

- So, the result stored in the String , you can do anything with it.
Eg: Manipulate it
```java
//Print it
System.out.println(result);

//You can make it Uppercase
Sout(result.toUpperCase())
```
