## Overview 
- For Each of the Primitive datatypes, there is a corresponding wrapper class.


The forllowing are Primitive Datatypes and their Wrapper Classes
i. byte -> Byte
ii. short -> Short
iii. int -> Interger
iv. long -> Long
v. float -> Float
vi. double -> Double
vii. boolean -> Boolean
viii. char -> Character

- The wrapper classes allows us to use the primitives as objects.
- Their benefit is seen when we start talking about Data Structures and generics.

Using the Wrapper Classes:
- The wrapper classes have the methods that you could use to manipulate the data.
ie:
```java
//int age = 20;  //Normal 

Integer age = 20; //Wrapper Class

var ageStr = "55";
int a = Interger.parseInt(ageStr); //Using the method that comes with the Wrapper Class
```

more methods in wrapper classes:
```java

Integer.min();  //Write the wrapper Classes followed by the dot and see the methods
```
