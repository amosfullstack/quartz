The main method is :
```java
public static void main(String[] args){
 //Code
}
```

i. <span style="color:rgb(255, 255, 0)">public</span> : public means that this method can be accessed publicly. That is, inside the class, outside the class, inside the package and outside the package.

ii. <span style="color:rgb(255, 255, 0)">static</span> : Means that there is no need of creating an instance of the class, the method can be invoked directly.

iii. <span style="color:rgb(255, 255, 0)">void</span> : Means that the method has no return type.

iv. <span style="color:rgb(255, 255, 0)">main</span> : This is the name of the main method.

v. <span style="color:rgb(255, 255, 0)">String args</span> : These are programme arguments. You can pass arguments here during the progrmme execution.


The main method is the special method needed by the JVM.
- The JVM needs the main method so as to start the application.
