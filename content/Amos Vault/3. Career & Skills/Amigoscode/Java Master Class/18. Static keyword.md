What does a static keyword mean?
- Any property declared with the static keyword, means that it belongs to the class and not the instance itself.
- If something is static it <span style="color:rgb(255, 255, 0)">belongs to the class</span>.

Therefore, the property can be called using the name of the class and the dot operator.
ie:
```java
public class Main{
  static String userName = "Amos";
}

//In other class
Main.userName; //The static Propertery is called
```

NB : Within a static context, you need to use something that is static.
Example : Withing a <span style="color:rgb(255, 255, 0)">static method</span>, you must use a <span style="color:rgb(255, 255, 0)">static varible</span>, otherwise nothing will work.
ie:
```java
public class Main{
   static String userName = "Amos";
   String brandName = "Lenovo";
   
   public static void main(String[] args){
   
      sout(userName); //This will Works
      sout(brandName); //This wont work, because brandName is not static
   }
}
```

To use something that is not static, we create its instance
ie:
```java
   Main m = new main();
   sout(m.brandName); //This will work
```
