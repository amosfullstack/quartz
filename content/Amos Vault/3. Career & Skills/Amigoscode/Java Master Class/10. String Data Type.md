Strings : Allows us to store a sequence of Characters.

ie:
```java
String name = "Amos Samwel";
String email = "amos@gmail.com";
```

#### String concatenation
- The plus sign +, is used to concatenate Strings
ie:
```java
String firstName = "Amos";
String lastName = "Samwel";
String fullName = firstName + lastName; //String concatenated
sout(fullName);
```
