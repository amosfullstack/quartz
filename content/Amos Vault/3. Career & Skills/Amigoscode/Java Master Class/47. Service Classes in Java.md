What is Business Logic?
- Refers to what the application is supposed to do in certain Scenarios.

What are Service Classes?
- The service classes contains the business Logic of Whatever problem you are trying to solve.
- Inside the Packages, there will be the <span style="color:rgb(255, 255, 0)">Service Class</span> for the Corresponding <span style="color:rgb(255, 255, 0)">Model Class</span> .

Examples of Business Logic:
- You want to include the Class of Storing the Cars to the Database.
- You want a Logic to Add the Cars to the Garage, also what happens when the Garage is full.
- You want a Class for the Logic of person Registation For the Person Class.
Eg :
  <span style="color:rgb(255, 148, 148)">com.amos.Person</span>
      ->Person.java (Model Class) : For Holding the Data
      ->PersonService.java (Serivice Class) : For Handling the Business Logic

ie:
```java
public class PersonService {  
    public int addPerson(Person person){  
        if(person.getFirstName().isBlank()){  
            throw new IllegalArgumentException(  
                    "FirstName can not be null or Empty"  
            );  
        }  
        //If it Works, store person to db  
        return 1;  
    }  
}
```
