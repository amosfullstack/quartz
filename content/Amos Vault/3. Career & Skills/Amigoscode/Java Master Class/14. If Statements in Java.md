If statements allows us to execute the block of code, based on a condition.
- The condition can either be <span style="color:rgb(255, 255, 0)">true</span> or <span style="color:rgb(255, 255, 0)">false</span> .
- If the condition is true the piece of code is executed, otherwise.. the other piece of code will be executed.
ie:
Syntax
```java
if(condition){
  //Execute this code if true
}
```

Also:
```java
if(condition){
  //Execute this code if true
}
else{
  //Execute this code if not true
}
```

Example:
```java
int age = 15;
boolean isAdult = age>=18;
if(isAdult){
   sout("He is an adult :)");
 }
 else{
   sout("He is not an adult:(");
 }
```

Using <span style="color:rgb(255, 255, 0)">else if</span> :
- Else if is used to add more condition to be checked
ie: 
```java
int age = 15;
boolean isAdult = age>=18;
if(isAdult){
   sout("He is an adult :)");
 }
 else if(age==1){//else if added condition to check
   sout("He is a toddler"); 
 }
 else{
   sout("He is not an adult:(");
 }
```

```java

}

}
```
