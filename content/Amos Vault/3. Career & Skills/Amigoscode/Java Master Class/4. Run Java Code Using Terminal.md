1. Select the Terminal icon in Intellij, this will open the command line.
2. Change directory to the source folder (src).
3. Execute the main class(To compile the java Program into intermediate byte code).
ie: 
```bash
javac com/Amos/Main.java

//Means the Main.java class is located in com.Amos Package
```

4. Run the bytecode created. (Main.class).
ei: 
```cli
java com/Amos/Main

//Alternative, you can use dot .
java com.Amos.Main
```
When running the class, you dont need the .class extension. It just knows that you want to run your class.

<span style="color:rgb(0, 176, 80)">NB</span> : The compiled classes are stored in the <span style="color:rgb(255, 255, 0)">out </span>folder in the Project structure in IntelliJ

#### To see the Bytecode that is fed to the JVM
- Go to the <span style="color:rgb(255, 0, 255)">out </span>folder.
- Select the class you want to view bytecode.
- On the tab menu, go to <span style="color:rgb(255, 0, 255)">view</span>.
- The Click <span style="color:rgb(255, 0, 255)">Show bytecode</span> . Bytecode the JVM understands will be displayed in the editor.
