#### Clear Explanation about the source code
i. Given: 
```java
class Main{
	  public static void main(String[] args){
	    //Code here
	  }
}
```

<span style="color:rgb(255, 255, 0)">public static void main</span> :
- The main method is very important. Is the one that allows the programme to run. Without it the programme can not run. It should be public.
- Whatever code that is inside the main function. Will be executed.
- It is the main entry point for your application.

ii. Every statement in Java should end with the semicolon. Without the semicolon the code will fail to compile.

ii. Download the plugin <span style="color:rgb(255, 255, 0)">rainbow brackets</span>
