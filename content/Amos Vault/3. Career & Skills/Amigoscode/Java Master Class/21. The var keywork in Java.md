What is the <span style="color:rgb(255, 255, 0)">var</span> keyword?
- The var keyword was introduced in <span style="color:rgb(255, 255, 0)">java 10</span> .
- It allows the<span style="color:rgb(255, 255, 0)"> automatic detection </span>of the variables being used.
- This concept is known as <span style="color:rgb(255, 255, 0)">type inference </span>.

ie:
```java 
//Type Inference with the var keyword

var name = "Amos";
var names = new String[]{"Amos","Daniel","Samwel"};
var isAdult = false;
var age = 24;
var balance = 1_000_000.33;

//Basically i do not to define a datatype when decaring a new variable
```

- The var keyword works for every data type, even if it is an object.

QN : Can you use the var keyword everywhere?
<span style="color:rgb(0, 176, 80)">Answer</span> : NO , You can only use the var keyword for local variables.
ie: if you have a method, and in that method you have variables, then you can use the var keyword in there.

- The var keyword will not work in the class level scope.
