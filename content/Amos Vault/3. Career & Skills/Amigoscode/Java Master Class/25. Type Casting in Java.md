What is Type Casting?
- Type Casting is the process of converting a datatype into another.
There are :
- Implicit Type Casting (Widening)
- Explicit Type Casting (Narrowing)


i. Implicit Type Casting.
- This involves converting a smaller datatype into a bigger data type.
Eg : Converting the interger into the double.
ie:
```java
int balance = 100;

double balanceInDouble = balance; //Type casted here.
```

ii. Explicit type Casting.
- This involves  converting the lager datatype to a smaller Data type.
Eg : Converting the double into the interger.
ie:
```java
double remainingBalance = 100.55;

int remainingBalanceInt = (int)remainingBalance; //Type casted here
```


NB : If your application really cares about decimal numbers, you have to be really careful when performing type casting to your primitives.
