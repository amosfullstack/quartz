What is a method?
- A method is a block of code that runs when it is <span style="color:rgb(255, 255, 0)">Invoked</span>.
- We can use methods to <span style="color:rgb(255, 255, 0)">reuse code</span> that can be shared across the entire application.


This to consider when declaring a method :
- Access modifier (public,private)
- Optional Static
- Return type (void,datatype)
- Name (name of the method)
- Optional Parameters (parameters that the method accepts)
- Method body (Code to be executed)
- Optional Retrun Value (What the method should return)


Practical : Creating a method that adds two numbers
```java
private static int add(int n1, int n2){
  int result = n1+ n2;
  return result;
}

```

- n1 and n2 are called parameters.
- The values of n1 and n2 that we pass when calling the method are called arguments.


#### Methods without Return type.

#### Naming Conventions of Methods in java
- Methods should be verbs in mixed case, with the first letter lower case, with the letter of each internal word capitalized.
Eg : run(); , runFast(); , getBackGround();
