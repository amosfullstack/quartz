## **Overview**
- Learning how to SELECT data from the database using the select query from the PHP website.
- Followed by inturn displaying the data on the website.

## **Steps to Follow**

###### 1. Create a php script that will contain the php code
- Name the script as <span style="color:rgb(255, 255, 0)">userselect.php</span> etc..

###### 2. Grab the data from the page tha was created for data search
- Use the same modality for data grabbing, create the varibales to store the values referenced by the name attribute.
###### 3. Just Update the Query to perform selection
```php
$query = "SELECT * FROM comments WHERE username = :usersearch;";
//Basically this selects all the rows in the comments table.
```
###### 4. Create prepared statement, bind parameters and execute statement
```php
$stmt = $pdo->prepare($query);
//i. Bind Parameters
$stmt->execute();
//Now things change here, afte execution you have to grab that selected data
```

###### 5. Fetch the data from the executed Statement and store them in the varible
```php
$results = $stmt->fetchAll(PDO::FETCH_ASSOC);

//All the data fetched from the database are stored in the variable $results as an associative array.
```

From the code snippet:
> <span style="color:rgb(255, 148, 148)">$stmt->fetchAll()</span> - t’s a **PDOStatement method** that retrieves **all the rows** returned by your query.
>                               - Instead of getting just one row (like <span style="color:rgb(255, 148, 148)">fetch()</span>), it collects **everything** into an array.
>                               - You can control the format with **fetch style constants**:
>                               - The fetch style contants can be used with both <span style="color:rgb(255, 148, 148)">fetchAll() </span>and <span style="color:rgb(255, 148, 148)">fetch().</span>
>                                  i. <span style="color:rgb(255, 255, 0)">PDO::FETCH_ASSOC </span>→ associative array only
                                    ii. <span style="color:rgb(255, 255, 0)">PDO::FETCH_NUM </span>→ numeric array only
                                    iii. <span style="color:rgb(255, 255, 0)">PDO::FETCH_OBJ </span>→ objects

In our case, the data is fetched as associative array because grabbing data from the database <span style="color:rgb(255, 0, 255)">associative arrays</span> becomes much easier. Beacause each array data will have the name associated to it, so we can reffer to the name in order to get the data.(<span style="color:rgb(255, 255, 0)">Key => Value)</span> implies to (<span style="color:rgb(255, 255, 0)">Name => Data</span>) imples to (<span style="color:rgb(255, 255, 0)">Column=>Actual Data</span>).
ie: column names in the database are the <span style="color:rgb(0, 176, 80)">name</span> in assoc array and values are <span style="color:rgb(0, 176, 80)">actual data.</span>
- **Keys** = column names (`id`, `username`, `email`)
- **Values** = actual data from the database (`1`, `"Amos"`, `"amos@example.com"`)

###### 6. HTML code Display results inside the Same file
i. Check if there is any data it the variable $results,if empty do not proceed.
```php
if(empty($results)){
  echo "<div>";
  echo "<p>There were no results fetched</p>";

```

ii. If not empty display the results from the array, we do not know how many data have been fetched so we are going to use the <span style="color:rgb(255, 255, 0)">foreach() </span>to loop over all the available data.
```php
else{

```
