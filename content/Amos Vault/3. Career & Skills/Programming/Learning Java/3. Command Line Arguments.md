Command line Arguments: Are data that are given to the main method.
- String[] args: This is a group of strings called arguments(args).
- We are able to give the main method a group of strings and we can use them.
Example:

```java
public class Main{

  public static void main(String[] args){
     sout(args[0]);
     sout(args[1]);
     sout(args[2]);
  }

}
```

The above is the Main.java file then:
```bash
javac Main.java 

//Execute and pass the arguments, the three argumenrs are passec
java Main hello 123 food
```

- You might never need to use command line arguments when building programs, but it is important to understand what is written over there.
