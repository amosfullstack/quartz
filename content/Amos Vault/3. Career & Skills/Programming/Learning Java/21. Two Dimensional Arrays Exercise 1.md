QUESTION
Write a programme that prints the sum of each row in a 2D Array.

Solution
```java
  
int[][] intergers = {  
        {2,4,5},  
        {3,4},  
        {6,4,3,9,0,4,5}  
};  
  
//Make sure the sum initialization is in the inner scope  
for(int i=0; i<intergers.length; i++){//Iterate all the rows  
    int sum = 0; //Initial sum  
    for(int j=0; j< intergers[i].length; j++ ){ //Interate the colums in the current row  
        sum+=intergers[i][j]; //Summation in each rows column  
    }  
    //Output after the the 3 rows iteration  
    System.out.println("Sum of row "+(i+1)+" = "+sum);  
}
```
