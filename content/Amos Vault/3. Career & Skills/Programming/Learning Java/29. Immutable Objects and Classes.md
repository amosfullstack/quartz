What is an Immutable Object?
- Is an Object whose contents can not be changed.
- Sometimes, we want to create an object whose contents can not be changed once an object has been created. Such an Object is Called an <span style="color:rgb(255, 0, 255)">Immutable Object</span> and its Class is an <span style="color:rgb(255, 0, 255)">Immutable Class</span>.

Example :
- The String Class is an Immutable Class and the String Objects are Immutable Objects.


How to Say that a Class is an Immutable Class.
- All the  Data Fields must be private.
- There cant be any Setter methods in the class.
- There Shoud be No getter methods can return a data field that is mutable.
      -> If there is a getter method that returns an object, this object should be immutable.

Example of Immutable Class.
```java
class C1 {
//Attribute
  private int x;
  
  //Constructor
  public C1(int x){
    this.x = x;
  }
  
  //Getter
  public int getX(){
     return this.x;
  }
}
```
- This is an Immutable Class, Because it Obeys All the Rules. eg No setters.
- Also all the data fields are private.
