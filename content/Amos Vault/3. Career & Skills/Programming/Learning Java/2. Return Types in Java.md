What are Return types?
- First of all they are types.
- They are the types of data that a method returns/gives.
- A method can return or give a value.

Example 1:
```java
getUserName();
```
- The above method,gets the name of the user and returns a value.
- The retutn type is a text(String).

Example 2:
```java
getUserAge();
```
- The above method,gets the age of the user and returns a value.
- The return type is a number (int ,double).


###### The void return type.
- main() has a void return type.
- void means nothing.
- So the main method does not return a value.

Example 1:
```java
printUserName();
```
- We just want to print the name of the user.
- We do not want to get any value from this method.
- The return type is void.
