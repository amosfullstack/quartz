###### Null Value
What is the null Value?
- Is the value of an object that references nothing.
ie :
```java
String text; //This is null
```
- Here the string variable will contain the address of the string, because there is no value assigned to it.
- So, If we create an object, without assigning any value to it. It will contain <span style="color:rgb(255, 255, 0)">null</span>.
- Null Means that the Object does not reference anything.


###### Arrays
What are arrays?
- Arrays is the <span style="color:rgb(255, 255, 0)">collection</span> of variables of the same data type.
- An array in Java is an <span style="color:rgb(255, 255, 0)">Object</span> .
- An array in java references a <span style="color:rgb(255, 255, 0)">group of data</span> or a collection of variables of the same data type .
- The size of array is <span style="color:rgb(255, 255, 0)">fixed</span>. So, the array has some size that can not be changed after we create an array.

Declaring an array in Java:
```java
dataType[] arrayName; //null

arrayName = new dataType[10]; //Address of 10 elements of dataType is Reserved

//All the above on a single line
dataType[] arrayName = new dataType[10];


//Example
int[] numbers = new int[10]; //The memory to store 10 intergers is Reserved.
```


Default Values :
 When an array is created , its elements are assigned the following default Values.
- <span style="color:rgb(255, 255, 0)">0</span> for numeric primitive data types.
- <span style="color:rgb(255, 255, 0)">\u0000 </span>for char types.
- <span style="color:rgb(255, 255, 0)">false</span> for boolean types .
- <span style="color:rgb(255, 255, 0)">null</span> for reference types.(Eg Strings,Points etc)


###### Accessing array Elements
- We shall use square brackets and indices.

eg:
```java
int[] numbers = new int[5]; //{0,0,0,0,0}

numbers[0]=5; //accessing the first Element of an array {5,0,0,0,0}


numbers[2]=8; //The third element is now equal to 8 {5,0,8,0,0}
 
```


###### Array Initializers
- Suppose we want to initialize the value of the element when we create an array.
eg:
```java
double[] numbers = {10.3,3.4,5.6,4.6,78.43}; //The array is declare an initialized
```


###### Printing Arrays

NB : Printing just the variable name of the array, will just print the address and not the array values.
ie:
```java
int[] numbers = {10,20,20,40,50};  
System.out.println(numbers); //Prints Address in the memory

//Output
[I@15aeb7ab
```

How to print the values of an array?
- Use a loop. ie:
```java
for(int i=0; i<numbers.length; i++){  
    System.out.print(numbers[i]+" ");  
}
```
