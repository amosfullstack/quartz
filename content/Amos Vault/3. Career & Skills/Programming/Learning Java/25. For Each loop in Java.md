### Explanation
What is a for each loop ?
- It is a  type of loop, that we can use to iterate over arrays and arraylists more easily.
Syntax :
```java
for( TYPE VAR_NAME : Arraylist/Array){
  ...
}
```

what happens:
- In each iteration, the variable VAR_NAME will hold the value of an element inside the arraylist/array. Starting from the first Element.
- There is no index used.
- We are safe, we can not exceed the boundaries of the arraylist or the array. Safe(Boundaries.)

Example :
Consider the fruits Arraylist.
```java
ArrayList<String> fruits = new ArrayList<>();  
  
//Add Elements  
fruits.add("Apple");  
fruits.add("Strawberry");  
fruits.add("Coconut");  
fruits.add("Banana");  
fruits.add("Mango");  
  
//Using a for each loop  
for(String fruit : fruits){  
    System.out.print(fruit+" ");  
}
```
- <span style="color:rgb(255, 0, 255)">fruit</span> : Is a declared varible to hold temporarily, the values of the array while looping through.
- <span style="color:rgb(255, 0, 255)">fruits</span> : Is the actual arraylist we are looping through.


Exercise :
- Create an Interger arraylist, then print its values contatenated with their index.
```java
ArrayList<Integer> numbers = new ArrayList<>();  
numbers.add(3);  
numbers.add(5);  
numbers.add(6);  
numbers.add(1);  
numbers.add(7);  
  
//Using a for each loop  
int i=0;   //Initalize i counter first
for(Integer number : numbers){  
    System.out.println(number+" "+i++);  //Intement it after each iteration
}
```

Alternative :
```java
//Using a for each loop  
int i=0;  
for(Integer number : numbers){  
    System.out.println(number+" "+i);  
    i++;   //i counter is incremented here
}
```
