###### Anonymous arrays
What are anonymous arrays?
- An array without a variable referencing to it.
eg:
```java
new int[] {10,20,20,10};
```

###### Exceeding Array bounds
- When we want to access the elements of an array, the indices must be between <span style="color:rgb(255, 255, 0)">0</span> and <span style="color:rgb(255, 255, 0)">length-1</span> 

eg:
```java
char chars = {'a','b','c','d'};

sout(chars[-1]); //This will throw an Exception
//Index -1 out of bounds for length 4

sout(chars[4]);
//Index 4 out of bounds for length 4
```
NB: You must be careful not to exceed the bounds of an array.
Otherwise : <span style="color:rgb(255, 255, 0)">ArrayIndexOutOfBoundException</span> shall be thrown


###### Passing Arrays to methods.
- Arrays are passed by reference, because arrays are Objects.

eg:
```java
int numbers = {0,1};
change(numbers); //The address of an array is passed as an argument to the method

//The method change
public static void change(int[] numbers){
  numbers[0] = 1; //{1,1} at this point
  numbers[1] = 0; //{1,0} at this point
}
```

###### Returning arrays from methods

```java
public static int[] getNumbers(){  
    int[] numbers = {1,2,3,4,5};  
    return numbers;  //This return the Address of the array
}
```

- To get the values from the array, reference to its address again
```java
public static void main(String[] args) {  
  
    int[] numbers = getNumbers();  //Referencing to the address that  stored the array
    for (int i=0; i<numbers.length; i++){  
        System.out.println(numbers[i]+" ");  
    }  
  
}
```
