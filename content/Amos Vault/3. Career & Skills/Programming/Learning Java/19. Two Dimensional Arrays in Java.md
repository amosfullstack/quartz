### Explanation
- Two Dimensional Arrays are used to represent data that look like a table or a matrix.
- A two dimensional array is a one dimensional array in which each element is another one dimensional array.

The 2d array syntax
```java
int [rows][colums] numbers; //null

numbers = new int[5][3];

//We can do all the above in one statement
int[][] numbers = new int[5][3]; //five rows, three colums.
```


Accesing the element at the certain index in the array.
- Use the index of the row and the index of the column.
```java
numbers[row][column];

//Accessing the Element at row 0 and column 0
numbers[0][0];
```


Example :
```java
int[][] numbers = new int[2][2]; //[[0,0][0,0]]  
  
numbers[0][0] = 3;  
numbers[0][1] = 5;  
numbers[1][0] = 10;  
numbers[1][1] = 2; //[[3,5],[10,2]]

// Doing it genereally
int[][] numbers = {{3,5},{10,2}};
```

iii. Printing the 2d array elements column by colums.
- The outer loop is for the columns and the inner loop is for the rows
