UML : Unified Modelling Language 
- The UML Language is used to Create Diagrams in CS.



Class Diagram Representation using UML.
- Using the UML Class Diagram, We can specify the <span style="color:rgb(255, 255, 0)">name</span>, <span style="color:rgb(255, 255, 0)">attributes</span> and <span style="color:rgb(255, 255, 0)">methods</span> of a Class
ie:

| ClassName                       |
| ------------------------------- |
| visibility attribute : type     |
| visibility method : return type |

Example  1:

| Cirlcle                        |
| ------------------------------ |
| -radius : double               |
| -*numberOfObjects* : int       |
| +Circle(double)                |
| +getArea() : double            |
| *+getNumberOfObjects : double* |
From the Class Diagram :
`-` Symbol  : Indicates that the attribute is the <span style="color:rgb(255, 255, 0)">private</span> Attribute. 
`+` Symbol : Indicates that the attribute is the <span style="color:rgb(255, 255, 0)">public</span> Attribute or method.
underlined/Itallic : Indicates the <span style="color:rgb(255, 255, 0)">Static</span> Attribute .
