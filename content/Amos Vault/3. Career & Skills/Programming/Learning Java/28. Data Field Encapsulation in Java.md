What is Data Field Encapsulation?
- This reffers to Preventing direct Modification of the datafields by declaring them as private, by using the private visibility modifier.


If the data fields are not Private
- This makes the class difficult to mantian and Vulnerable To Bugs.


What is done.
- The data fields are declared as private.
- Thus we are not able to access them using a dot operator.
- To be able to access the private data field we shall use the <span style="color:rgb(255, 0, 255)">getters</span> and the <span style="color:rgb(255, 0, 255)">setters</span>.
