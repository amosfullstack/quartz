### Explanations

What is an Inner Class?
- Is a class inside another Class.
- Just like we have the if Statement inside other if Statement.

There are two types of Inner Classes.
i. Member Class.
ii. Static Member Class.


The Properties of the Inner Class
i. The Inner Class can access the Private members of the Containing Class.
ii. The inner Class can have the private access modifier so that, it is restricted to one class only.

Example :
```java
class Outer{
    
    class Inner {
//Attribute
  private int x;
  
  //Constructor
  public C1(int x){
    this.x = x;
  }
  
  //Getter
  public int getX(){
     return this.x;
  }
}
}
```
From Above:
- The Inner Class is a <span style="color:rgb(255, 0, 255)">member</span> Class.
- NB : The Inner Class cannot contain the static variables.
- Since the Inner Class will be associated to the instances of the Outer Class, so it can not have <span style="color:rgb(255, 148, 148)">static</span> Variables.
- But, the Inner Class can contain the <span style="color:rgb(255, 255, 0)">final static variables</span> .
