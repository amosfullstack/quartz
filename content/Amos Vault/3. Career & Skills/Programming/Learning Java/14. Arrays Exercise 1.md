Exercise 1:
a. Write a program that fills an array with n, intergers Entered by the user.

b. Suppose the user can enter atleast 1 number and at most 20 numbers.

c. Resolve the same Excercise with Points.

Solution a and b:
```java
  
public class Main {  
    public static void main(String[] args) {  
        //Instantiate the Scanner Class  
        Scanner input = new Scanner(System.in);  
  
        //Enter the number of Elements of the Array  
        System.out.print("How many Elements? Max(20): ");  
        int n = input.nextInt();  
  
        //Make Sure the array size is greater than 0 and less than 20  
        while(n>20 || n<=0){  
            System.out.print("Invalid input,try again!");  
            n = input.nextInt();  
        }  
  
        //Initialize the array  
        int[] numbers = new int[n];  
  
        //Fill the Array  
        fillArrayOfIntergers(numbers);  
        printArrayOfIntergers(numbers);  
    }  
  
    private static void printArrayOfIntergers(int[] numbers) {  
        System.out.print("The elements are: ");  
        System.out.println(Arrays.toString(numbers));  
    }  
  
    private static void fillArrayOfIntergers(int[] numbers) {  
        Scanner input = new Scanner(System.in);  
        System.out.println("Enter the Array elements.");  
        for(int i=0; i<numbers.length; i++){  
            System.out.print("Element "+(i+1)+" :");  
            numbers[i] = input.nextInt();  
        }  
  
    }  
}
```


Solution for c :
```java
public class Main {  
    public static void main(String[] args) {  
        //Instantiate the Scanner Class  
        Scanner input = new Scanner(System.in);  
  
        //Enter the number of Elements of the Array  
        System.out.print("How many Elements? Max(20): ");  
        int n = input.nextInt();  
  
        //Make Sure the array size is greater than 0 and less than 20  
        while(n>20 || n<=0){  
            System.out.print("Invalid input,try again!");  
            n = input.nextInt();  
        }  
  
        //Initialize the array  
        Point[] points = new Point[n];  
  
        //Fill the Array  
        fillArrayOfPoints(points);  
        printArrayOfPoints(points);  
    }  
  
    private static void printArrayOfPoints(Point[] points) {  
        System.out.println("The points are: ");  
        for(int i=0; i<points.length; i++) {  
            System.out.print("Point "+(i+1)+": ");  
            System.out.println("(" + points[i].x+","+points[i].y+")");  
        }  
    }  
  
    private static void fillArrayOfPoints(Point[] points) {  
        Scanner input = new Scanner(System.in);  
        System.out.println("Enter the Array points.");  
        for(int i=0; i<points.length; i++){  
            System.out.print("Enter x and y for point "+(i+1)+" :");  
            points[i] = new Point(input.nextInt(), input.nextInt());  
        }  
  
    }  
}
```
