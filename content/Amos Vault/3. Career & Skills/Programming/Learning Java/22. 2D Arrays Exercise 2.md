QUESTION :
Write a programme that prints the maximum of each row of the 2D array.

Soln :
```java
public static void main(String[] args) {  
  
  int[][] intergers = {  
          {8,4,5},  
          {3,4},  
          {6,4,3,9,0,4,5},  
          {-3,4,0,3,4,5}  
  };  
  
  //Make sure the max is in the inner scope  
  for(int i=0; i<intergers.length; i++){//Iterate all the rows  
      int max = intergers[i][0]; //Initialize the maximum number to be 1st element  
      for(int j=1; j< intergers[i].length; j++ ){ //Interate the colums in the current row  
          max = (intergers[i][j]>max)?  intergers[i][j] : max;  
      }  
      //Output after the the 3 rows iteration  
      System.out.println("Maximum of row "+(i+1)+" = "+max);  
  }  
  
}
```
