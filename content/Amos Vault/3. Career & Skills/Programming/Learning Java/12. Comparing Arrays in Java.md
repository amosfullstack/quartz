###### Comparing Arrays
Suppose we have three Arrays.
```java
int[] numbers1 = {1,2,3,4,5};  
int[] numbers2 = {1,2,3,4,5};  
int[] numbers3 = {2,3,4,1,7};
```

i. Using the comparison operator ==
```java
System.out.println(numbers1==numbers2); //false
```
- Returns false becase we are comparing the address of numbers1 and numbers2, their addresses are completely different.

ii. Using the <span style="color:rgb(255, 255, 0)">equals()</span> of the Arrays Class
```java
System.out.println(Arrays.equals(numbers1,numbers2)); //true

System.out.println(Arrays.equals(numbers1,numbers3)); //false, arrays are not equal
```
- Here Java is comparing the <span style="color:rgb(255, 255, 0)">contents</span> of the arrays and not the addressed
- If all elements at the same indexes are equal it will return true, if atlest one elements are not equal it will return false.

iii. It the same to compare the Arrays of Objects.
ie:
```java
String[] letters1 = {"a","b","c"};  
String[] letters2 = {"a","b","c"};  
  
System.out.println(Arrays.equals(letters1,letters2)); //true
```


iv. Comparing the arrays of point Objects.

```java
Point[] points1 = {new Point(1,2),new Point(3,4)};  
Point[] points2 = {new Point(1,2),new Point(3,4)};  
  
System.out.println(points1==points2); //false  (Wrong Way)
System.out.println(Arrays.equals(points1,points2)); //true (Right Way)
```

NB : There is an equals method for objects already built in Java. It Specifies how we can compare objects of the Same type.
- All Objects should be compared using the <span style="color:rgb(255, 255, 0)">.equals()</span> method.
- REMEMBER : Strings are Objects.
