QUESTION:

Write a program that displays the number of occurences of an element in the Array.

Answer :
```java

  
public class Main {  
    public static void main(String[] args) {  
        //Declare an array  
        int[] numbers = {2,3,5,6,7,8,9,2,1,1,3};  
  
        //Declare the Element to be searched  
        int searchElement = 1;  
  
        System.out.println(getSearchOccurance(numbers,searchElement));  
  
    }  
  
    //Method to find the occurences of a number  
    public static int getSearchOccurance(int[] numbers,int searchElement){  
        //Create counter variable to store occurences  
        int occ = 0;  
        for(int i=0; i< numbers.length; i++){  
            if(searchElement==numbers[i]){  
                occ++;  
            }  
        }  
        return occ;  
    }
    
```
