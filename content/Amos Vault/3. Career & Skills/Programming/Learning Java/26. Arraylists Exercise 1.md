QUESTION :
i. Create a list of  Elements Takes from the user.Sort and Print these Elements as unique.
- Even if the user Enters dupilcate Elements, the output should be sorted and unique.Thus we did not store duplicate elements.

SOLUTION :
a. Read n elements from the user.
b. If the element does not exist in the arraylist add it. (If we have already stored the element in the arraylist, we will not store it the second time.)
c. Sort the arraylist using Collections.sort();

```java
ArrayList<Integer> numbers = new ArrayList<>();  
  
System.out.println("Enter 5 numbers:");  
Scanner input = new Scanner(System.in);  
for(int i=0; i<5; i++){  
    int n = input.nextInt();  
  
    if(!numbers.contains(n)){  //Use the contains method of the arraylist 
        numbers.add(n);  
    }  
}  
  
Collections.sort(numbers); //Use the sort method of the Collections class
System.out.println(numbers);
```
