QN :
Write a programme that displays, the sum, average and product of an interger array.


Solution :
```java
  
public class Main {  
    public static void main(String[] args) {  
  
        int[] numbers = {1,2,3,4};  
  
        int sum = 0;  
        int product = 1;  
        double average;  
  
        for(int i=0; i<numbers.length; i++){  
            sum += numbers[i];  
            product *= numbers[i];  
        }  
  
        average = sum/numbers.length;  
  
        System.out.println("Sum :"+sum);  
        System.out.println("Product :"+product);  
        System.out.println("Average :"+average);  
  
    }
```
