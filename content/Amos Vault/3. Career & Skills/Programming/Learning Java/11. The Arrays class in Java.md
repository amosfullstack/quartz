###### The Arrays Class
What in the Arrays Class?
- It is a class in Java that contains some static methods that are used with arrays.

Using the methods of this class we can:
- Sort arrays.
- Search arrays.
- Compare arrays.
- Fill arrays.
- Return the string Representation of an array.

#### SORTING ARRATS IN JAVA
- Suppose you want to sort arrays in the increasing order.
- Use the <span style="color:rgb(255, 255, 0)">sort()</span>  method of the Arrays Class.
- Use the sort method and pass an array as the parameter.
ie:
i. Sorting Intergeres
```java
sort(array) //sorts the whole array

//EXAMPLE
int[] numbers = {10,20,4,5,-1,0,5,4};  
Arrays.sort(numbers); //-1 0 4 4 5 5 10 20
```
NB : The sort() method modifies the original array. This is because the arrays are passed by reference.
```java
for (int i = 0; i < numbers.length; i++) {  
    System.out.print(numbers[i]+" ");  //We are printing the same original array but already sorted.
}
```

ii. Sorting Characters.
- The arrays sort()  method can sort characters.
- What happens, the capital leters are sorted first and then followed by small letters.
eg;
```java
char[] letters = {'Q','W','Z','B','C','X','D','C','a','x','c','b'};  
Arrays.sort(letters);

//Sorted Array
B C C D Q W X Z a b c x 
```
NB : Characters are sorted based on their unicodes.

iii. Sorting an array for index to index
```java
sort(array, fromIndex, toIndex);
```

iii. Sorting Strings
```java
String[] names = {"amos","daniel","atugonza","eliakim","gilly"};  
Arrays.sort(names);

//Sorted array
amos atugonza daniel eliakim gilly 

```


#### SEARCHING ARRAYS IN JAVA
- Suppose we have an array and we want to search for an element in the array.
- We use the <span style="color:rgb(255, 255, 0)">binarySearch() </span> method of the Arrays Class.

Steps :
- The arrays should be <span style="color:rgb(255, 255, 0)">sorted</span> in increasing order.
- Then <span style="color:rgb(255, 255, 0)">binarySearch</span>(<span style="color:rgb(255, 148, 148)">array</span>, <span style="color:rgb(255, 148, 148)">element</span>)
eg:
```java
binarySearch(numbers, 4); //We are searching for the interger 4, in the array numbes
```

- Return values of the binarySearch() method : 
      -The <span style="color:rgb(255, 255, 0)">index of the element</span> inside the array if the element being searched exists.
      -<span style="color:rgb(255, 255, 0)">-(insertionIndex+1)</span>  if the element was not found in the array.(Negative value of the insertion Index plus one)

What is the insertionIndex?
- The insetion index, is the index where we insert an element and the Array remains sorted.
Example: We have an array {1,2,3,5,6,7}
- Where is 4 supposed to be inserted and the array to remain sorted
              : {1,2,3,4,5,6,7} 4 inserted and array remain sorted
- So, the insertion index, is the index of that 4, over there. The index is 3(the method will return <span style="color:rgb(255, 255, 0)">-4</span>)

Example:
```java
int[] numbers = {-1,3,4,5,2,1,0,9};  
Arrays.sort(numbers);  

System.out.println(Arrays.toString(numbers));//[-1, 0, 1, 2, 3, 4, 5, 9] sorted

  
System.out.println(Arrays.binarySearch(numbers,9));  //7 index , no found
System.out.println(Arrays.binarySearch(numbers,5));  //6 index, no found
System.out.println(Arrays.binarySearch(numbers,-4)); //-1 insertionIndex, no notfound 
}
```
