i. Method that gets a 2D array as a Parameter and the method that retuns a 2D array
ie:
```java
public static void main(String[] args) {  
  int[][] numbers = getArray();  
  printArray(numbers);  
}  
  
//The Method that Returns an array  
public static int[][] getArray(){//Notice the return type  
    return new int[][] {{2,3,5},{5,6,8},{4,2,9}}; //Returns anonymous array  
}  
  
//The method that takes an Array as the Parameter  
public static void printArray(int[][] numbers){  
     for(int i=0; i<3; i++){  
         for(int j=0; j<3; j++){  
             System.out.print(numbers[i][j]+" ");  
         }  
         System.out.println();  
     }  
}
```

Alternative :
```java
  
//The method that takes an Array as the Parameter  
public static void printArray(int[][] numbers){  
     for(int i=0; i<numbers.length; i++){ //numbers.length gives the number of rows  
         for(int j=0; j<numbers[i].length; j++){ //numbers[i].length gives the no of colums  
             System.out.print(numbers[i][j]+" ");  
         }  
         System.out.println();  
     }  
}
```


### Ragged Arrays
- Is the 2D array of the rows of different Length.
- Remember a 2D Array is an array of array, So each array can have its Own Length.
Consider :
```java
int[][] intergers = {  
        {2,4,5},  
        {3,4},  
        {6,4,3,9,0,4,5}  
};
```
In the above array :
- <span style="color:rgb(255, 0, 255)"> integers.length</span> = 3 : Becase we have 3 arrays in our 2D array.
- <span style="color:rgb(255, 0, 255)">intergers[i].length</span> : This will give the length of each array separately .
> ie : In first array = 3
