###### Filling Arrays.
- Whenever we create an array, the default values in <span style="color:rgb(255, 255, 0)">null</span>.
- In order to fill an array, We can use a <span style="color:rgb(255, 255, 0)">loop</span>.
- Also we can use a .<span style="color:rgb(255, 255, 0)">fill()</span> method of the Arrays class.

i. Using the <span style="color:rgb(255, 255, 0)">fill()</span> method
ie: Syntax
```java
//1.
//fill(array,value) : fill the whole array


//2.
//fill(array, fromIndex, toIndex, value) : fill the array in the specified indexed
```

Eg :
```java
int[] numbers = new int[3]; //{0,0,0}  
Arrays.fill(numbers,3); //{3,3,3} the entire array is filled
```

Eg2:
```java
int[] numbers = new int[5]; //{0,0,0,0,0}  
Arrays.fill(numbers,2,4,3); //{0,0,3,3,0} from index 2 to before 4
```


###### Printing Arrays
 - Use the .<span style="color:rgb(255, 255, 0)">tostring()</span> method of the Arrays Class.
 - This returns the string representation of an array.
ie:
```java
System.out.println(Arrays.toString(numbers));
```
