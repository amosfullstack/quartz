QUESTION :

Write a programme that displays the minimum and the maximum elements of an array.


Answer :

```java
public static void main(String[] args) {  
   //Create an array  
    int[] numbers = {2,3,4,5,6,7,-3};  
  
    //Assume that the first Element is the maximum and the minimum  
    int max = numbers[0];  
    int min = numbers[0];  
  
    //Iterate the array to find the new values of max and min  
    //Start from the second Element, because the first is already assigned    for(int i=1; i<numbers.length; i++){  
        max = (numbers[i]>max)? numbers[i] : max;  
        min = (numbers[i]<min)? numbers[i] : min;  
    }  
  
    System.out.println("min = "+min);  
    System.out.println("max = "+max);  
}
```
