## Outline
- Learn to Loop throught an Arraylist.
- Learn to Sort an Arraylist.

i. Looping throught an ArrayList.
- We can use a for loop.
ie: 
```java
//Looping through an ArrayList  
for (int i = 0; i <fruits.size(); i++) {  
    System.out.print(fruits.get(i)+" ");  
}
```


ii. Sorting the ArrayList
- Use the <span style="color:rgb(255, 0, 255)">sort()</span>  of the <span style="color:rgb(255, 0, 255)">Collections</span> class for sorting lists alphabetically or nurmerically.
ie:
```java
System.out.println(fruits);  
Collections.sort(fruits);  //The sort method of Collections Class
System.out.println(fruits); //Fruits will be sorted Alphabetically


//Output
[Apple, Strawberry, Coconut, Banana, Mango] //Unsorted
[Apple, Banana, Coconut, Mango, Strawberry] //Sorted

```

- The sort() method can sort an Arraylist alphabetical and numerically.
