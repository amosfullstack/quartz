- The Above keywords are used in object oriented Programming in Java.

1. Public  keyword
- This is called the public access modifier.
- It specifies how to access classes,methods and fields.
- The public keyword means that the access level is everywhere.

That means that eveything that is public,can be accessed:
- Inside the class.
- Outside the class.
- Inside the package.
- outside the package.

2. Private keyword
- This is called the private access modifier.
- If we use private, that means that the access level is only inside the class.

2. Static keyword
- This is a non access modifier.
- Using the static keyword, means that you will be able to access the methods and fields using the classname.
ie: 
```java
System.out
```
From above:
> We are accessing the static field <span style="color:rgb(255, 0, 255)">out</span>, using its classname <span style="color:rgb(255, 0, 255)">System</span>.
> - Out is the static field of system.
> - If it is not a static field we can not be able to access it using the name of the class.
