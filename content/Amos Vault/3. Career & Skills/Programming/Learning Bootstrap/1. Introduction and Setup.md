i. What is Bootstrap(5)?
- Boostrap is   a <span style="color:rgb(255, 0, 255)">Frontend</span> <span style="color:rgb(255, 255, 0)">CSS</span> and <span style="color:rgb(255, 255, 0)">Javascript</span> Framework for making mobile-first , <span style="color:rgb(255, 255, 0)">responsive</span> websites.
- It includes pre-made components like <span style="color:rgb(255, 255, 0)">navbars</span>,<span style="color:rgb(255, 255, 0)">modals</span>,<span style="color:rgb(255, 255, 0)">tabs</span>,<span style="color:rgb(255, 255, 0)">tooltips</span>, <span style="color:rgb(255, 255, 0)">buttons</span>, <span style="color:rgb(255, 255, 0)">accordions</span> and many more.
- Also comes with the responsive , 12-Column, CSS grid system for layouts. It can be used to distribute content on our webpage.
- That means we can write interactive websites without writing a single line of CSS or Javascript.
- Easy to customize. also ensures browser compatibility (for supported browser)

ii. How to Use Bootstrap:
Option 1:
- Dowload the <span style="color:rgb(255, 0, 255)">Compiled CSS and JS </span>and include it it the project folder.
- Use the CDN(Content Deliverly Network) links for both CSS and JS.
- Use <span style="color:rgb(255, 0, 255)">npm</span> to install it locally into our project.


iii. We are going to use CDN  for now:
- Place the CSS at the head of the html page.
- Place the Javascript at the end of the body of the page.

iv. Previewing the Page:
- Install the <span style="color:rgb(255, 0, 255)">Live Server </span>Extension to preview the page.
