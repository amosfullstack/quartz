Bootstrap uses a <span style="color:rgb(255, 255, 0)">Reboot</span>.
- That means, It removes all the default colors, marigins, fonts and typography etc that the browser provides.
- It replaces them with a set of basic bootstrap styles.
- You can see all of these styles into the boostrap documentation.
- When you start using bootstrap, you just start using the basic styles provided by bootstrap.
