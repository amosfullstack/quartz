Check boxes and radio buttons use the same classes for everything.

i. Peforming basic styling for check boxes and radio buttons.

```html
<form>
	
	<div>
		<input type="checkbox" id="checkbox1" class="form-check-input">
		<label for="checkbox1" class="form-check-label">Check Box</label>
	</div>
	
	<div>
		<input type="radio" id="radio1" class="form-check-input">
		<label for="radio1" class="form-check-label">Radio</label>
	</div>

</form>
```

- add <span style="color:rgb(255, 0, 255)">form-check-input</span> in the input tags : Styles a bit both the checkboxes and the radios.
- add <span style="color:rgb(255, 0, 255)">form-check-label</span> in the label tags : Makes the label aware, it does not style much.
- add <span style="color:rgb(255, 0, 255)">form-check</span> nto the wrapper divs of the Check boxes and the radios.


ii. Turn the checkbox or Radio into the toggle button.
- add <span style="color:rgb(255, 0, 255)">form-switch</span> class into the wrapper div
ie:
```html
<div class="form-check form-switch">
```

iii. Make two or more  checkboxes or radios get side by side, instead  of getting stacked on top of each other.
- add <span style="color:rgb(255, 0, 255)">form-check-inline</span> class inside the wrapper divs.
ie:
```html
<div class="form-check form-switch form-check-inline">
```


iv. Put the check boxes on the opposite side of the page.
- add <span style="color:rgb(255, 0, 255)">form-check-reverse </span>inside the wrapper divs.
ie:
```html
<div class="form-check form-switch form-check-reverse">
```
