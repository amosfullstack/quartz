- These Extras did not fit into any category.
i. File input
ii. Range
iii. Text input

Consider the code below
```html
<form>

	<div class="mb-3">
		<label class="form-label" for="Input1">Input Text</label>
		<input class="form-control" type="text" id="input1" disabled>//booean atttibute added
	</div>

  

	<div class="mb-3">
		<label class="form-label" for="file1">File Input</label>
		<input class="form-control" type="file" for="file1">
		<div class="form-text">Please select a file</div>	
	</div>

  

	<div class="mb-3">
		<label class="form-label" for="range1">Range</label>
		<input type="range" id="range1" class="form-range ">
	</div>

</form>
```


- add <span style="color:rgb(255, 0, 255)">form-range</span> into range input tag: This styles the range input into more appealing way.
- add <span style="color:rgb(255, 0, 255)">form-control</span> into file input tag : The style of file selection files become bombastic.
- add <span style="color:rgb(255, 0, 255)">form-text</span> : This gives small grey discription for what is to be filled in the file place.


Boolean Attributes to add on text input
- <span style="color:rgb(255, 255, 0)">readonly</span> : makes the input field undeitable.
- <span style="color:rgb(255, 255, 0)">disabled</span> : makes the input field unclickable.
