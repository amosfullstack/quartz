## **Overview**
- Bootstrap has many components that use the ordered List and Unordered List.
- They can be styled to make a bootstrap, List group component.


i. The fancy way of Displaying the Ordered and Unorderd List in Bootstrap
ie:
```html
<h6>Components with a List Tag</h6>

<ul class="list-group">

<li class="list-group-item list-group-item-action list-group-item-success">One</li>

<li class="list-group-item list-group-item-action list-group-item-success">Two</li>

<li class="list-group-item list-group-item-action list-group-item-success">Three</li>

</ul>
```
- <span style="color:rgb(255, 0, 255)">list-group</span> : Adds/introduces the list group component to the list. ie: removes the margins
- <span style="color:rgb(255, 0, 255)">list-group-item</span> : Styles the list Items into the table with borders.
- <span style="color:rgb(255, 0, 255)">list-group-item-action</span> : Makes the list items respond on hover .
- <span style="color:rgb(255, 0, 255)">list-group-item-success</span> : Provedes colors to the list items. Also provides the appropriate color on hover

Example 2:
- Removing the border stylings on the List Items
```html
<ul class="list-group list-group-flush list-group-numbered">

	<li class="list-group-item list-group-item-action">One</li>
	<li class="list-group-item list-group-item-action">Two</li>
	<li class="list-group-item list-group-item-action">Three</li>

</ul>
```
- <span style="color:rgb(255, 0, 255)">list-group-flush </span>: Removes the borders around the lists.
- <span style="color:rgb(255, 0, 255)">list-group-numbered</span> : Makes the ordered/unordered list be numbered.
This list styles can be applied on any html elements, either divs,buttons,spans etc


ii. The <span style="color:rgb(255, 255, 0)">breadcrumb</span> component
- The other component that utilizes the ul/ol tag is the breadcrum component.
```html
<ul class="breadcrumb">

	<li class="breadcrumb-item"><a href="#">Home</a></li>
	<li class="breadcrumb-item"><a href="#">Products</a></li>
	<li class="breadcrumb-item"><a href="#">Contacts</a></li>
	<li class="breadcrumb-item">About Us</li>

</ul>
```
- <span style="color:rgb(255, 0, 255)">breadcrumb</span> : Adds the breadcrumb component, it immediatly puts content of the list into one line.
- <span style="color:rgb(255, 0, 255)">breadcrumb-item</span> : Separates the contents of the breadcrumb, it comes with built in separatos ie: <span style="color:rgb(255, 255, 0)">foward slashes</span>
So: To customize or put the divider you would like:
```html
<ul class="breadcrumb" style="--bs-breadcrumb-divider:'* or any'">
```


iii. The <span style="color:rgb(255, 255, 0)">nav</span> component.
- It also uses the ol/ul tags.
- The ol and ul can be used interchangebly becaus, bootstrap strips away all the styles.
ie:
```html
<ul class="nav">

	<li class="nav-item"><a class="nav-link" href="#">Home</a></li>
	<li class="nav-item"><a class="nav-link" href="#">Products</a></li>
	<li class="nav-item"><a class="nav-link"href="#">Contacts</a></li>
	<li class="nav-item"><a class="nav-link" href="#">About Us</a></li>

</ul>
```
- <span style="color:rgb(255, 0, 255)">nav</span> : Adds the nav component
- <span style="color:rgb(255, 0, 255)">nav-item</span> : Makes the nav items aware.
- <span style="color:rgb(255, 0, 255)">nav-link</span> : Styles the links in the navigation bar.
Hence, the basic navigation bar becomes ready.

Add actions on the navigation bar.
- make one of the link tab active.
```html
<li class="nav-item"><a class="nav-link active" href="#">Products</a></li>
```

- make sure to add first the property in the main list
```html
<ul class="nav nav-underline">

or
<ul class="nav nav-pills" >
or
<ul class="nav nav-tabs">
```
- <span style="color:rgb(255, 0, 255)">nav-undeline</span> : Makes the active tab be underlined.
- <span style="color:rgb(255, 0, 255)">nav-pills </span>: Makes the active tab look like a basic button. 
- <span style="color:rgb(255, 0, 255)">nav-tabs </span>:  Makes the nav lok like the tapped folders.

There are other classes that deals with spacing of the nav bar.
ie:
```html
<ul class="nav nav-fill" >
or
<ul class="nav nav-justified">
```
- <span style="color:rgb(255, 0, 255)">nav-fill</span> : The navigations space any size to cover the entire container ie the body
- <span style="color:rgb(255, 0, 255)">nav-justified</span> : The nav components are of the same width across the page.


iv. The <span style="color:rgb(255, 255, 0)">pagination</span> component.
- This is used of multipage arrangement to enable easy navigation.
ie:
```html
<ul class="pagination">

	<li class="page-item"><a class="page-link" href="#"><<</a></li>
	<li class="page-item"><a class="page-link" href="#">1</a></li>
	<li class="page-item"><a class="page-link active"href="#">2</a></li>
	<li class="page-item"><a class="page-link" href="#">3</a></li>
	<li class="page-item"><a class="page-link" href="#">4</a></li>
	<li class="page-item"><a class="page-link" href="#">>></a></li>

</ul>
```
- <span style="color:rgb(255, 0, 255)">pagination</span> : Adds the pagination component to the ordered list.
- <span style="color:rgb(255, 0, 255)">page-item</span> : Makes the list items be aware that they are page items.
- <span style="color:rgb(255, 0, 255)">page-link </span>: Styles the page navigation links.

Use the <span style="color:rgb(255, 255, 0)">active</span> class to indicate the active page.
ie:
```html
<li class="page-item"><a class="page-link active"href="#">2</a></li
```

Use the <span style="color:rgb(255, 255, 0)">disabled</span> class to make the page unclickable.
```html
<li class="page-item"><a class="page-link disabled"href="#">2</a></li
```
