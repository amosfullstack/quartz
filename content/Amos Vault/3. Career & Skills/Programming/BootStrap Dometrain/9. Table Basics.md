i. The table <span style="color:rgb(255, 255, 0)">component</span> 
- This styles the table in bootstrap way, buy just adding the <span style="color:rgb(255, 255, 0)">table</span> class inside the table tag.
- This component introduces spacing and horizontal lines.

ie: Consider the structure of the table, with the table component
```html
<table class="table">

<thead>

	<tr>
		<th>#</th>
		<th>Name</th>
		<th>Age</th>
	    <th>Course</th
	</tr>

</thead>

<tbody>

	<tr>
		<td>1</td>
		<td>Amos</td>
		<td>24</td>
		<td>SE</td>	
	</tr>

</tbody>

</table>
```
- <span style="color:rgb(255, 0, 255)">table</span> : This class has introduces horizontal line and spacing between the contents.

ii. Make the line that separate the thead and tbody thicker.
- add <span style="color:rgb(255, 0, 255)">table-group-divider</span> class in the tbody tag
ie:
``` html
<tbody class="table-group-divider">
```

iii. Make the table smaller , make it more compact.
- add the <span style="color:rgb(255, 0, 255)">table-sm</span> class in the table tag
ie:
```html
<table class="table table-sm">
```
