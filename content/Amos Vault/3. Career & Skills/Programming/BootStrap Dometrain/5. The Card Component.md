What is a <span style="color:rgb(255, 255, 0)">card</span> component?
- It is a flexible and exitensible content container. It includes options for headers and footers, a wide variety of contents, and powerful display options.


ie: 
```html
<div class="card w-50">

	<div class="card-header text-bg-warning">Form One</div>
	
	  <img src="images/Screenshot from 2025-08-27 12-17-34.png">
	
		  <div class="card-body">
		    <h5 class="card-title">Girls School</h5>
		    <h6 class="card-subtitle">Pretty</h6>
		    <p class="card-text">
		         Lorem ipsum dolor reprehenderit aperiam.
		    </p>
		  </div>
	
	<div class="card-footer">Footer</div>

</div>
```
- <span style="color:rgb(255, 0, 255)">card</span>  : Adds the card component
- <span style="color:rgb(255, 0, 255)">text-bg-warning</span> : changes the color of the card background to Yellow (Warninh)
- <span style="color:rgb(255, 0, 255)">w-50</span> : Resizes the card component to take width 50%, becase it can take the entire seize of the container
