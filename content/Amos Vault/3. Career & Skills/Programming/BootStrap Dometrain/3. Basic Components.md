1. Basic <span style="color:rgb(255, 255, 0)">button</span> component
```html
<button type="button" class="btn btn-primary">Basic Button</button>
```
- <span style="color:rgb(255, 0, 255)">btn</span> : Adds the button component .
- <span style="color:rgb(255, 0, 255)">btn-primary</span> : specifies the color of the button .
Colors : There are Eight colors that We can specify in Bootstrap
       i. Primary - Blue
       ii. Secondary - Gray
       iii. Success - Green
       iv. Warning - Yellow
       v. Danger - Red
       vi. Info - Blue
       vii. Light - White
       viii. Dark - Black

2. The <span style="color:rgb(255, 255, 0)">margin</span> component
- This specifies the margin. It can be given the numbers from 1 to 5
- The margin is the <span style="color:rgb(255, 255, 0)">space around</span> an Object
ie:
```html
<body class="m-3">  //The body will have the margin of 3
<body>
```


3. The <span style="color:rgb(255, 255, 0)">badge</span> component
- This can be used for things like notifications.Its like a badge.
ie:
```html
<button type="button" class="btn btn-primary">Notifications

<span class="badge bg-info rounded-pill">8</span>

</button>
```

- <span style="color:rgb(255, 0, 255)">badge</span> : Adds the badge component.
- <span style="color:rgb(255, 0, 255)">bg-info</span> : class that Changes the color of the background(<span style="color:rgb(255, 0, 255)">bg</span>) to <span style="color:rgb(255, 0, 255)">info</span> .
- <span style="color:rgb(255, 0, 255)">rounded-pill</span> :  Class that makes a badge into round .

4. The <span style="color:rgb(255, 255, 0)">spinner</span> Component
- This is used for <span style="color:rgb(255, 0, 255)">loading</span> purposes.
ie:
```html
<span class="spinner-grow"></span>Loading //Looks like a growing sound
//or
<span class="spinner-border spinner-border-sm"></span>Loading  //Looks like a normal loading we know
```
- <span style="color:rgb(255, 0, 255)">spinner-border-sm</span> : Adding this class makes the spinner small

5. The button group component.
- The button group component collapses multiple buttons into one block.
- It removes the space between the buttons.
ie:
```html
<div class="btn-group"> //button group component added here

    <button class="btn btn-primary">Choice 1</button>
	<button class="btn btn-primary">Choice 2</button>
	<button class="btn btn-primary">Choice 3</button>

</div>
```
- <span style="color:rgb(255, 0, 255)">btn-group</span> : This class adds the button group component to an Element.

6. The <span style="color:rgb(255, 255, 0)">button</span> outline component.
- This colors the outerlines of the button, instead of filling it with the color.
```html
<button class="btn btn-outline-info">Choice 1</button>
```

To shouw the selected button use the <span style="color:rgb(255, 255, 0)">active</span> class
ie:
```html
<div class="btn-group">

	<button class="btn btn-outline-info">Choice 1</button>
	<button class="active btn btn-outline-info">Choice 2</button> //active btn
	<button class="btn btn-outline-info">Choice 3</button>

</div>
```
- Other button on hover, will use the same color that was used on the active button.

7. The <span style="color:rgb(255, 255, 0)">alert</span> Component.
 - This is used to style Warnings and Popups on the webpage.
 ie:
 ```html
 <div class="alert alert-danger">
      <span>Invalid Operation!</span>
</div>
 ```
 - <span style="color:rgb(255, 0, 255)">alert</span> : Adds the alert component.
 - <span style="color:rgb(255, 0, 255)">alert-danger</span> : Sets the color of the alert component to danger(red)

If we want to make the alert able to be dismissed
```html
<div class="alert alert-danger alert-dismissible">

	<span>Invalid Operation!</span>
	<button class="btn-close"></button>

</div>
```
- <span style="color:rgb(255, 0, 255)">btn-close</span> : adds an X at the end of the alert, it can be clicked to dismiss the alert.
- <span style="color:rgb(255, 0, 255)">alert-dismissible</span> : Puts the X on the correct position, such that it is at the end of the alert.
