What is a CSS framework?
- A CSS framework is a pre-defined library of CSS or Styles.
- It helps speed up the development process and reduces the styling complexity.
- There are so many CSS propeties, it is not easy to memorize them all, thats when CSS framework comes in.

Two Popular Frameworks:
- Bootstrap
- Tailwind CSS
- There are other CSS goodframe works, but they are just less known.
