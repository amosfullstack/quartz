
___

This is also called the <span style="color:rgb(255, 0, 255)">progress bar</span>.

Eg:
Creating the progress bar.
```html
<div class="progress mb-2">
	<div class="progress-bar w-25"></div>
</div>	

<div class="progress">
	<div class="progress-bar w-75"></div>
</div>

```
- <span style="color:rgb(255, 0, 255)">progress</span> : Adds a progress wrapper, ie: where the progress bar will live
- <span style="color:rgb(255, 0, 255)">mb-2</span> : Adds the margin botton to 2px.
- <span style="color:rgb(255, 0, 255)">progress-bar</span> : creates the progress bar.
- <span style="color:rgb(255, 0, 255)">w-25</span> : Makes the progress width be 25%

You can make a progress bar have the <span style="color:rgb(255, 255, 0)">stripes</span> look.
```html
<div class="progress-bar progress-bar-striped w-75"></div>
```
- <span style="color:rgb(255, 0, 255)">progress-bar-striped </span>: Makes the progress bar have the zebra stripes.

You can also make the progress bar <span style="color:rgb(255, 255, 0)">animated</span>.
```html
<div class="progress-bar progress-bar-striped progress-bar-animated w-75"></div
```
- <span style="color:rgb(255, 0, 255)">progress-bar-animated</span> : Makes it look like the stripes are moving

You can change the <span style="color:rgb(255, 255, 0)">color</span> of the progress bars
```html
<div class="progress-bar bg-danger progress-bar-striped w-75"></div>
```
- <span style="color:rgb(255, 0, 255)">bg-danger</span> : Changes the color of the progress bar to danger(red)

You can add text inside the progress bar
```html
<div class="progress mb-2">

	<div class="progress-bar bg-danger progress-bar-animated w-25">
	  25%  //The text will be seen in the progress bar
	</div>

</div>
```


You can stack the progress bar together such that they are on one line.
ie:
```html
<div class="progress-stacked">

	<div class="progress w-25">
	    <div class="bg-danger progress-bar progress-bar-striped"></div>
	</div>

	<div class="progress w-50">
	   <div class="bg-success progress-bar progress-bar-striped"></div>
	</div>

</div>
```
- <span style="color:rgb(255, 0, 255)">progress-stacked</span> : The wrapper that puts bothe progress bars on one line, like a stack. On top of another.
