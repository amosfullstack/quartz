Considering the basic form structure.
```html
<form>

<div class="mb-3">
	<label class="form-label" for="input1">Input Field</label>
	<input class="form-control" type="text" id="input1">
</div>

  

<div class="mb-3">

	<label class="form-label" for="select1">Select Field</label>
	
		<select class="form-select" id="select1">
			<option>Option 1</option>
			<option>Option 2</option>
			<option>Option 3</option>
			<option>Option 4</option>
		</select>

</div>

		<button type="button" class="btn btn-primary">Button</button>

</form>
```

- <span style="color:rgb(255, 0, 255)">form-control </span>: This class is added in the input field, It nakes the input block level.
- <span style="color:rgb(255, 0, 255)">form-select</span> : The class added to the select field, it makes the select field block level and styles it.
- <span style="color:rgb(255, 0, 255)">form-label</span> : It styles the label tag and adds some margins and spacing.

i. Make the input field small or large.
- add form-control-sm int the input tag , for small.
- add form-contror-lg into the input tag, to make it large
ie:
```html
//for small
<input class="form-control form-control-sm" type="text" id="input1">

//For large
<input class="form-control form-control-ls" type="text" id="input1">
```

Also, works the same for the select tag.
```html
//for small
<select class="form-select form-select-sm" id="select1">

//for larger
<select class="form-select form-select-lg" id="select1">
```


ii. Creating the floating lables.
- Write the label tags <span style="color:rgb(255, 255, 0)">after</span> the input field.
- add the <span style="color:rgb(255, 0, 255)">placeholder</span> attribute in the input field.
- add the <span style="color:rgb(255, 0, 255)">form-floating</span> class in the wrapper div.
ie:
```html
<div class="mb-3 form-floating">

	<input class="form-control" type="text" id="input1">
	<label class="form-label" for="input1">Input Field</label>

</div>
```
