## Overview
- Bootstrap Comes with the natural style for form Validation.
- But browsers now have thier own form validation styling.
- Inorder to use the bootstrap styling we must turn off the browser styling.

Step 1. To turn off the browser styling, add the <span style="color:rgb(255, 0, 255)">novalidate</span> boolean attribute to the form tag
ie:
```html
<form novalidate> //Turn off the default validation of the browsers

</form>
```

Step 2: Add the required boolean attribute to the input fields
