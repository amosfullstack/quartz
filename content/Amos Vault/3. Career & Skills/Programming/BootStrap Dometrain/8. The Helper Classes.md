What are Helper Classes used for?
- Changing the color, eg <span style="color:rgb(255, 255, 0)">background</span> color.
- Used to change the <span style="color:rgb(255, 255, 0)">text</span> color.
- Changing the <span style="color:rgb(255, 255, 0)">width</span> of the component .

eg :
i. Changing the background color of the text using helper class
```html
<div class="w-50 text-bg-warning">

	<p>Lorem ipsum dolor, sit amet consectetur adipisicing elit. Quidem nam cumque ut veritatis delectus laborum possimus quas fugiat laudantium qui. A necessitatibus quam quisquam, optio distinctio consequatur doloribus tempore reiciendis.</p>

</div>
```
- w-50 : Makes the width take 50%.
- text-bg-warning : Sets the background color of the div to <span style="color:rgb(255, 255, 0)">warning</span>  ti comes with the default color of the text, which is black

ii. To set the color of the text separately
```html
<div class="w-50 text-white bg-warning">

	<p>Lorem ipsum dolor, sit amet consectetur adipisicing elit. Quidem nam cumque ut veritatis delectus laborum possimus quas fugiat laudantium qui. A necessitatibus quam quisquam, optio distinctio consequatur doloribus tempore reiciendis.</p>

</div>
```
- <span style="color:rgb(255, 0, 255)">text-white</span> : Makes the text color to white
- <span style="color:rgb(255, 0, 255)">bg-warning</span> : Makes the background color to warning.


ii. Setting the color and the hover color of the link Tags
```html
<a class="link-danger" href="#">Language</a>
```
- <span style="color:rgb(255, 0, 255)">link-danger</span> : Sets the link color to danger


iii. Limit the text that is visible within a confined container.
- We use the <span style="color:rgb(255, 255, 0)">text-truncate</span> helper class.
```html
<div class="w-75 text-truncate text-info ">

	Lorem ipsum dolor sit amet consectetur adipisicing elit. Qui, eaque. Maxime ipsum temporibus, voluptatum blanditiis accusantium amet consequuntur impedit minus placeat veniam magnam expedita doloremque quibusdam exercitationem neque suscipit facilis.

</div>
```
- <span style="color:rgb(255, 0, 255)">text-truncate</span> : This class makes the text cover the container widh and other text is hidden.


iii. Introducing the separator line between texts |.
- Use the <span style="color:rgb(255, 255, 0)">vertical row</span> helper class.
ie:
```html
<div class="w-75 text-truncate text-info ">

Lorem ipsum <span class="vr"></span> sit amet consectetur adipisicing elit. Qui, eaque.

</div>
```
- <span style="color:rgb(255, 0, 255)">vr</span> : This introduces a line |


iv. The <span style="color:rgb(255, 255, 0)">placeholder</span> helper class.
 - Takes the width of the entire component and fills it.
 ```html
 <div class="w-75 text-truncate text-info placeholder">

		Lorem ipsum <span class="vr"></span> sit amet consectetur adipisicing            elit. Qui, eaque.

</div>
 ```
 - <span style="color:rgb(255, 0, 255)">paceholder</span> : Will replace everything and just introduce a solid block
