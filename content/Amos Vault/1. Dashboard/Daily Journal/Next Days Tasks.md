###### Downloaded Ventory (Rufus Alternative for Ubuntu)
- This is the best tool available today, on any operating system. You install it to a USB drive once, and then you can simply **drag and drop** any number of ISO files (Ubuntu, Windows, memtest, etc.) onto the USB drive. No need to reformat and rewrite the entire USB for each new ISO.
- Used to make the usb drive bootable, you can add multiple os to it.
```bash
#Download ventory and extract
tar -xzf ventoy-*.linux.tar.gz

#Run the script against the usbdrive to make bootable
cd ventoy-*
sudo ./Ventoy2Disk.sh -i /dev/sdX

```

**Why choose Ventoy?**
- Use one USB drive for all your ISOs.
- Incredibly easy to add or remove ISOs (just copy/delete files).
- Supports both BIOS and UEFI systems.
- Supports hundreds of different ISO types.
