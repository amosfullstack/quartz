## Description

Java Streams API is a Declarative programming Java programming course published by AmigosCode Academy. Declarative programming is a programming paradigm or coding style that is in stark contrast to imperative programming. The amount of code written in this style is less than other existing styles, and the programmer is able to execute far more complex commands with less code. In recent years, functional programming has gained a lot of fans and is used in a wide range of projects. Applications developed with functional programming have a smaller code size and their behavior can be predicted to some extent.

Functional programming has many advantages over object-oriented programming. For example, in this style of programming we will never encounter shared state disruption. shared state is a variable that is accessed by more than one function and it is possible for its value to be changed by other functions in a completely unwanted and unexpected way. Transferring from one programming style to another is by no means an easy task, but the instructor of this course has made every effort to be able to move you from grammatical programming to declarative programming and with unlimited capabilities and capacities. Familiarize its number.
