## The basic JavaScript tutorial

JavaScript is the implementation of ECMA Script (pronounced ECMA Script), which is also called ES, and ES6, in fact, version 6 of this standard, also called ECMA Script 2015 (the time of finalization of the standard is 2015) and is also the biggest change since the release of JavaScript. With ES6, things like classes, functional programming, modules, etc. have been added. Over the next 2 years, all new JavaScript frameworks, from Bootstrap 4 to AngularJS 2 to Aurelia, etc., will all move to ES6 (and that’s happening now). So soon, without learning and mastering the ES6, you will be considered illiterate in the web, and learning it is a must.
