### Question Three

3. With the aid of a well sketch, show the execution flow of the following recursive function. (5 Marks)  
   ```cpp
   #include<iostream>
   using namespace std;
   int facto(int);
   int main()
   {
       int num = 4;
       int factonum = facto(num);
       cout << "The factorial of " << num << " is: " << factonum;
   }
   int facto(int nam)
   {
       if(nam == 0)
           return 1;
       else
           return nam * facto(nam - 1);
   }
   ```
